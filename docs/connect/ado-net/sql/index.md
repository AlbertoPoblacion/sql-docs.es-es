# <a name="sql-server-and-adonet"></a>SQL Server y ADO.NET

[!INCLUDE[Driver_ADONET_Download](../../../includes/driver_adonet_download.md)]

## <a name="microsoftdatasqlclient"></a>Microsoft.Data.SqlClient

<xref:Microsoft.Data.SqlClient> proporciona acceso a versiones de SQL Server, que encapsula los protocolos específicos de base de datos. <xref:Microsoft.Data.SqlClient> incluye un analizador de flujos de datos tabulares (TDS) para comunicarse directamente con SQL Server.  
  
> [!NOTE]
> Para usar el proveedor de datos SqlClient de Microsoft para SQL Server, una aplicación debe hacer referencia al espacio de nombres <xref:Microsoft.Data.SqlClient>.
