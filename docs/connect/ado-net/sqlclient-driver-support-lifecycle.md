# <a name="sqlclient-driver-support-lifecycle"></a>Ciclo de vida de soporte del controlador SqlClient

![Download-DownArrow-Circled](../../ssdt/media/download.png)[Descargar ADO.NET](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Cada iteración del proveedor de datos SqlClient de Microsoft para SQL Server se suministra con su plataforma asociada. A continuación se muestran los detalles del ciclo de vida:

## <a name="microsoft-support-for-net-core"></a>Soporte técnico de Microsoft para .NET Core

|  Versión  |  Fecha de la versión | Nivel de soporte técnico | Versión de revisión compatible | Finalización del soporte |
| -- | -- | -- | -- | -- |
| [.NET Core 2.2](https://aka.ms/netcore22announce) | 4 de diciembre de 2018 | Current | [2.2.6](https://www.microsoft.com/net/download/dotnet-core/runtime-2.2.6) | |
| [.NET Core 2.1](https://blogs.msdn.microsoft.com/dotnet/2018/05/30/announcing-net-core-2-1) | 30 de mayo de 2018 | LTS | [2.1.12](https://www.microsoft.com/net/download/dotnet-core/runtime-2.1.12) | Al menos tres años a partir de la declaración de LTS (21 de agosto de 2018). |

## <a name="microsoft-support-for-net-framework"></a>Soporte técnico de Microsoft para .NET Framework

El ciclo de vida del producto de .NET Framework se puede ver [en la página](https://support.microsoft.com/lifecycle/search?alpha=.net%20framework) .
