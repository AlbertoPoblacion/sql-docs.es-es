# <a name="ole-db-driver-for-sql-server-programming"></a>Programación del controlador OLE DB para SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  OLE DB Driver for SQL Server es una API COM de bajo nivel que se usa para acceder a los datos. El controlador OLE DB para SQL Server es la opción recomendada para desarrollar herramientas, utilidades o componentes de bajo nivel que necesitan un alto rendimiento. El controlador OLE DB para SQL Server es un proveedor nativo de alto rendimiento con acceso directo al protocolo Flujo TDS de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 OLE DB Driver for SQL Server proporciona compatibilidad con OLE DB para aplicaciones que se conectan a [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 OLE DB Driver for SQL Server es un proveedor compatible con la versión 2.0 de OLE DB.
