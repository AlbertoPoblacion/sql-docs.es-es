# <a name="nodejs-driver-for-sql-server"></a>Controlador Node.js de SQL Server

![Download-flecha abajo: círculo](../../ssdt/media/download.png)[para descargar el controlador SQL de node. js](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

El módulo tedioso es una implementación Java del protocolo TDS, que es compatible con todas las versiones actuales de SQL Server. El controlador es un proyecto de código abierto que está disponible en github.  
  
Puede conectarse a un SQL Database mediante node. js en Windows, Linux o Mac.
