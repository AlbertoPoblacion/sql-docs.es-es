# <a name="datetimeoffset-class"></a>Clase DateTimeOffset
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este tipo representa un objeto de tipo microsoft.sql.DateTimeOffset y es el tipo de Java que se corresponde con el tipo de datos **datetimeoffset** de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 **Paquete:** microsoft.sql  
  
 **Extiende:** java.lang.Object  
  
 **Implementa:** java.lang.Comparable\<DateTimeOffset>, java.io.Serializable  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final class DateTimeOffset  
```  
  
## <a name="remarks"></a>Observaciones  
 Para más información sobre los tipos de datos, consulte [Descripción de los tipos de datos del controlador JDBC](../../../connect/jdbc/understanding-the-jdbc-driver-data-types.md).
