# <a name="createnclob-method-sqlserverconnection"></a>Método createNClob (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un objeto NClob sin datos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.sql.NClob createNClob()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Objeto NClob.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método createNClob se especifica mediante el método createNClob en la interfaz java. SQL. Connection.
