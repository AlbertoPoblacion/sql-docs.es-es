# <a name="length-method-sqlservernclob"></a>Método length (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número de caracteres del objeto **NClob**.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 La longitud de **NClob** en número de caracteres.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método de longitud especificado por el método length de la interfaz java.sql.NClob.
