# <a name="gettypemap-method-sqlserverconnection"></a>Método getTypeMap (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el objeto Map asociado con este objeto [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md).  
  
> [!NOTE]  
>  El [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)] no admite este método actualmente.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.util.Map getTypeMap()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto Map.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Observaciones  
 El método getTypeMap especifica este método getTypeMap en la interfaz java.sql.Connection.
