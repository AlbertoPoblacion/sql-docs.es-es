# <a name="createstatement-method-sqlserverconnection"></a>Método createStatement (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un objeto [SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md) para enviar instrucciones SQL a la base de datos.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[createStatement ()](../../../connect/jdbc/reference/createstatement-method.md)|Crea un objeto SQLServerStatement para enviar instrucciones SQL a la base de datos.|  
|[createStatement (int, int)](../../../connect/jdbc/reference/createstatement-method-int-int.md)|Crea un objeto SQLServerStatement que genera objetos SQLServerResultSet con el tipo y simultaneidad especificados.|  
|[createStatement (int, int, int)](../../../connect/jdbc/reference/createstatement-method-int-int-int.md)|Crea un objeto SQLServerStatement que genera objetos SQLServerResultSet con el tipo, simultaneidad y capacidad de alojamiento especificados.|
