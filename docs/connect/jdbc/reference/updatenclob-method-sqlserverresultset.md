# <a name="updatenclob-method-sqlserverresultset"></a>Método updateNClob (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Actualiza la columna designada con el valor de objeto especificado.

|Nombre|Descripción|  
|----------|-----------------|  
|[Método updateNClob &#40;int, java.sql.NClob&#41;](../../../connect/jdbc/reference/updatenclob-method-int-java-sql-nclob.md)|Actualiza la columna designada con un valor **NClob**.|  
|[Método updateNClob &#40;java.lang.String, java.sql.NClob&#41;](../../../connect/jdbc/reference/updatenclob-method-java-lang-string-java-sql-nclob.md)|Actualiza la columna designada con un valor **NClob**.|  
|[Método updateNClob &#40;int, java.io.Reader&#41;](../../../connect/jdbc/reference/updatenclob-method-int-java-io-reader.md)|Actualiza la columna designada mediante el objeto **Reader** especificado.|  
|[Método updateNClob &#40;int, java.io.Reader, long&#41;](../../../connect/jdbc/reference/updatenclob-method-int-java-io-reader-long.md)|Actualiza la columna designada mediante el objeto **Reader** especificado, cuya longitud queda establecida según el número de caracteres especificado.|  
|[Método updateNClob &#40;java.lang.String, java.io.Reader&#41;](../../../connect/jdbc/reference/updatenclob-method-java-lang-string-java-io-reader.md)|Actualiza la columna designada mediante el objeto **Reader** especificado.|
