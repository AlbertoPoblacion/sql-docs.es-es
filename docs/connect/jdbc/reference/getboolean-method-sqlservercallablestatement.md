# <a name="getboolean-method-sqlservercallablestatement"></a>Método getBoolean (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

## <a name="overload-list"></a>Lista de sobrecargas  
  
|NOMBRE|Descripción|  
|----------|-----------------|  
|[getBoolean (int)](../../../connect/jdbc/reference/getboolean-method-int.md)|Recupera el valor del parámetro designado como un valor **boolean** según el índice del parámetro.|  
|[getBoolean (java.lang.String)](../../../connect/jdbc/reference/getboolean-method-java-lang-string.md)|Recupera el valor del parámetro designado como un valor **boolean** según el nombre del parámetro.|
