# <a name="getstring-method-sqlservercallablestatement"></a>Método getString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un objeto **String** en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|NOMBRE|Descripción|  
|----------|-----------------|  
|[getString (int)](../../../connect/jdbc/reference/getstring-method-int.md)|Recupera el valor del parámetro designado como un objeto **String** en el lenguaje de programación Java según el índice del parámetro.|  
|[getString (java.lang.String)](../../../connect/jdbc/reference/getstring-method-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto **String** en el lenguaje de programación Java según el nombre del parámetro.|
