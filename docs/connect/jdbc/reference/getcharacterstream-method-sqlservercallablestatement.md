# <a name="getcharacterstream-method-sqlservercallablestatement"></a>Método getCharacterStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un objeto java.io.Reader.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[getCharacterStream &#40;int&#41;](../../../connect/jdbc/reference/getcharacterstream-int.md)|Recupera el valor del parámetro designado como un objeto java.io.Reader según el índice del parámetro.|  
|[getCharacterStream &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getcharacterstream-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto java.io.Reader según el nombre del parámetro.|
