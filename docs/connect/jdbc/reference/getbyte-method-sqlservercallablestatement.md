# <a name="getbyte-method-sqlservercallablestatement"></a>Método getByte (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un valor **byte**.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|NOMBRE|Descripción|  
|----------|-----------------|  
|[getByte (int)](../../../connect/jdbc/reference/getbyte-method-int.md)|Recupera el valor del parámetro designado como un valor **byte** según el índice del parámetro.|  
|[getByte (java.lang.String)](../../../connect/jdbc/reference/getbyte-method-java-lang-string.md)|Recupera el valor del parámetro designado como un valor **byte** según el nombre del parámetro.|
