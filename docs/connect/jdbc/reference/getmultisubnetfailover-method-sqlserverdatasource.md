# <a name="getmultisubnetfailover-method-sqlserverdatasource"></a>getMultiSubnetFailover (método) (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve el valor de la propiedad de conexión **multiSubnetFailover**.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
public boolean getMultiSubnetFailover();  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Devuelve true o false, dependiendo de la configuración actual de la propiedad de conexión.  
  
## <a name="remarks"></a>Observaciones  
 Para más información sobre la propiedad de conexión **multiSubnetFailover**, consulte [Establecimiento de las propiedades de conexión](../../../connect/jdbc/setting-the-connection-properties.md).
