# <a name="getasciistream-javalangstring"></a>getAsciiStream (java.lang.String)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un flujo de caracteres **ASCII** según el nombre del parámetro.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final java.io.InputStream getAsciiStream(java.lang.String paramName)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *paramName*  
  
 Un valor **String** que indica el nombre del parámetro.  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto InputStream.
