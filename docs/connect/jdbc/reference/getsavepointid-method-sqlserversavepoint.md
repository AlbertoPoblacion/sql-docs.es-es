# <a name="getsavepointid-method-sqlserversavepoint"></a>Método getSavepointId (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtiene el identificador del punto de retorno.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getSavepointId()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un **int** valor.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método getSavepointId especificado por el método getSavepointId en la interfaz java.sql.Savepoint.
