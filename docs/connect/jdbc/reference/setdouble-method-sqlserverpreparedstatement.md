# <a name="setdouble-method-sqlserverpreparedstatement"></a>Método setDouble (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado en el valor **double** indicado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setDouble(int n,  
                            double x)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *n*  
  
 Valor **int** que indica el número de parámetro.  
  
 *x*  
  
 Valor **Double** .  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 El método setDouble especifica este método setDouble en la interfaz java.sql.PreparedStatement.
