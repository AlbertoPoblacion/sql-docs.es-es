# <a name="createclob-method-sqlserverconnection"></a>Método createClob (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un objeto Clob sin datos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.sql.Clob createClob()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Objeto CLOB.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método createClob se especifica mediante el método createClob en la interfaz java. SQL. Connection.  
  
 Este método reemplaza la necesidad del [constructor &#40;SQLServerClob SQLServerConnection, Java. lang. String&#41;](../../../connect/jdbc/reference/sqlserverclob-constructor-sqlserverconnection-java-lang-string.md).
