# <a name="getcolumncount-method-sqlserverresultsetmetadata"></a>Método getColumnCount (SQLServerResultSetMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve el número de columnas del conjunto de resultados.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getColumnCount()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un valor **int** que indica el número de columnas.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Observaciones  
 El método getColumnCount especifica este método getColumnCount en la interfaz java.sql.ResultSetMetaData.
