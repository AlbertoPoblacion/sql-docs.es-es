# <a name="getint-method-sqlservercallablestatement"></a>Método getInt (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como **int** en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[getInt (int)](../../../connect/jdbc/reference/getint-method-int.md)|Recupera el valor del parámetro designado como un objeto **int** en el lenguaje de programación Java según el índice del parámetro.|  
|[getInt (java.lang.String)](../../../connect/jdbc/reference/getint-method-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto **int** en el lenguaje de programación Java según el nombre del parámetro.|
