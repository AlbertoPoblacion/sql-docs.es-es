# <a name="updateobject-method-sqlserverresultset"></a>Método updateObject (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Actualiza la columna designada con un valor de tipo **Object**.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[updateObject (int, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object.md)|Actualiza la columna designada con un valor **Object** según el índice de la columna.|  
|[updateObject (int, java.lang.Object, int)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object-int.md)|Actualiza la columna designada con un valor **Object** según el índice y la escala de columna.|  
|[updateObject (java.lang.String, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object.md)|Actualiza la columna designada con un valor **Object** según el nombre de columna.|  
|[updateObject (java.lang.String, java.lang.Object, int)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object-int.md)|Actualiza la columna designada con un valor **Object** según el nombre y la escala de columna.|
