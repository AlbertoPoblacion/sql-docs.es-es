# <a name="setbytes-method-sqlserverblob"></a>Método setBytes (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Escribe la matriz de bytes determinada en el BLOB comenzando en la posición determinada y posteriormente devuelve el número de bytes escritos.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[setBytes (long, byte&#91;&#93;)](../../../connect/jdbc/reference/setbytes-method-long-byte.md)|Escribe la matriz de bytes determinada en el BLOB comenzando en la posición determinada y posteriormente devuelve el número de bytes escritos.|  
|[setBytes (long, byte&#91;&#93;, int, int)](../../../connect/jdbc/reference/setbytes-method-long-byte-int-int.md)|Escribe todo o parte de la matriz de bytes determinada en el BLOB comenzando en la posición, desplazamiento y longitud determinados; y posteriormente devuelve el número de bytes escritos.|
