# <a name="getjdbcminorversion-method-sqlserverdatabasemetadata"></a>Método getJDBCMinorVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número de versión secundaria de JDBC para este controlador.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getJDBCMinorVersion()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un valor **int** que indica la versión secundaria de JDBC.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Observaciones  
 El método getJDBCMinorVersion especifica este método getJDBCMinorVersion en la interfaz java.sql.DatabaseMetaData.
