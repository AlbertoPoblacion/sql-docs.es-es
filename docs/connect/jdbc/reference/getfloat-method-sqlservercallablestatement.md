# <a name="getfloat-method-sqlservercallablestatement"></a>Método getFloat (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como **float** en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[getFloat (int)](../../../connect/jdbc/reference/getfloat-method-int.md)|Recupera el valor del parámetro designado como un objeto **float** en el lenguaje de programación Java según el índice de parámetro.|  
|[getFloat (java.lang.String)](../../../connect/jdbc/reference/getfloat-method-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto **float** en el lenguaje de programación Java según el nombre de parámetro.|
