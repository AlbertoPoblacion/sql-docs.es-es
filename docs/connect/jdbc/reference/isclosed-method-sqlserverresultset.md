# <a name="isclosed-method-sqlserverresultset"></a>Método isClosed (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Indica si se ha cerrado este objeto [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public boolean isClosed()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 **True** si este [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto está cerrado, **false** si aún está abierta.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método isClosed especificado por el método isClosed en la interfaz java.sql.ResultSet.
