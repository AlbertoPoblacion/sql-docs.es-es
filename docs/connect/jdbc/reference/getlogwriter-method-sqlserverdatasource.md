# <a name="getlogwriter-method-sqlserverdatasource"></a>Método getLogWriter (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este método solo se utiliza de forma interna. Para más información sobre el registro, consulte [Seguimiento del funcionamiento del controlador](../../../connect/jdbc/tracing-driver-operation.md).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.io.PrintWriter getLogWriter()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto PrintWriter.  
  
## <a name="remarks"></a>Observaciones  
 El método getLogWriter especifica este método getLogWriter en la interfaz javax.sql.DataSource.
