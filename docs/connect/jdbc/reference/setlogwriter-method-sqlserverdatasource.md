# <a name="setlogwriter-method-sqlserverdatasource"></a>método setLogWriter (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este método solo se utiliza de forma interna. Para más información sobre el registro, consulte [Seguimiento del funcionamiento del controlador](../../../connect/jdbc/tracing-driver-operation.md).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setLogWriter(java.io.PrintWriter out)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *out*  
  
 Un objeto PrintWriter.  
  
## <a name="remarks"></a>Observaciones  
 El método setLogWriter especifica este método setLogWriter en la interfaz javax.sql.DataSource.
