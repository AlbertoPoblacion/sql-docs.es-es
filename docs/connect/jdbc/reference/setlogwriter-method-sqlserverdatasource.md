# <a name="setlogwriter-method-sqlserverdatasource"></a>método setLogWriter (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este método solo se utiliza de forma interna. Para obtener más información sobre el registro, vea seguimiento de la [operación del controlador](../../../connect/jdbc/tracing-driver-operation.md).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setLogWriter(java.io.PrintWriter out)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *out*  
  
 Objeto PrintWriter.  
  
## <a name="remarks"></a>Notas  
 Este método setLogWriter se especifica mediante el método setLogWriter en la interfaz javax. SQL. DataSource.
