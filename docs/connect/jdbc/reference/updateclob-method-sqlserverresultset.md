# <a name="updateclob-method-sqlserverresultset"></a>Método updateClob (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Actualiza la columna designada con un valor java.sql.Clob.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[updateClob (int, java.sql.Clob)](../../../connect/jdbc/reference/updateclob-method-int-java-sql-clob.md)|Actualiza la columna designada con un valor java.sql.Clob según el nombre del índice.|  
|[Método updateClob &#40;int, java.io.Reader&#41;](../../../connect/jdbc/reference/updateclob-method-int-java-io-reader.md)|Actualiza la columna designada mediante el objeto Reader especificado.|  
|[Método updateClob &#40;java.lang.String, java.io.Reader, long&#41;](../../../connect/jdbc/reference/updateclob-method-java-lang-string-java-io-reader-long.md)|Actualiza la columna designada mediante el objeto Reader especificado, cuya longitud queda establecida según el número de caracteres especificado.|  
|[updateClob (java.lang.String, java.sql.Clob)](../../../connect/jdbc/reference/updateclob-method-java-lang-string-java-sql-clob.md)|Actualiza la columna designada con un valor java.sql.Clob según el nombre de la columna.|  
|[Método updateClob &#40;java.lang.String, java.io.Reader&#41;](../../../connect/jdbc/reference/updateclob-method-java-lang-string-java-io-reader.md)|Actualiza la columna designada mediante el objeto Reader especificado.|  
|[Método updateClob &#40;java.lang.String, java.io.Reader, long&#41;](../../../connect/jdbc/reference/updateclob-method-java-lang-string-java-io-reader-long.md)|Actualiza la columna designada mediante el objeto Reader especificado, cuya longitud queda establecida según el número de caracteres especificado.|
