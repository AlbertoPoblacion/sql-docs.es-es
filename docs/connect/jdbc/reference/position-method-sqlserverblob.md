# <a name="position-method-sqlserverblob"></a>Método position (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve la posición de un patrón especificado en el BLOB según el patrón determinado y el índice de inicio.

|[posición (byte&#91;&#93;, Long)](../../../connect/jdbc/reference/position-method-byte-long.md)|Devuelve la posición de un patrón especificado en el BLOB según el patrón de matriz **byte** determinado y el índice de inicio.|
