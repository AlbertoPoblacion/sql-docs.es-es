# <a name="python-sql-driver"></a>Controlador Python para SQL

![En círculo la flecha abajo de descarga](../../ssdt/media/download.png)[para instalar el controlador SQL para Python](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Puede conectarse a una base de datos de SQL con Python en Windows, Linux o Mac.   
  
## <a name="getting-started"></a>Introducción  
Hay varios controladores de SQL Server de python. Sin embargo, Microsoft pone sus esfuerzos de pruebas y su confianza en **pyodbc** controlador. Elegir un controlador y configurar el entorno de desarrollo, según corresponda:
* [Controlador Python para SQL: pyodbc](pyodbc/python-sql-driver-pyodbc.md)
* [Controlador Python para SQL: pymssql](pymssql/python-sql-driver-pymssql.md)
