# <a name="odbc-test"></a>Prueba de ODBC
Microsoft® Prueba ODBC es una aplicación habilitada para ODBC que puede usar para probar controladores ODBC y el Administrador de controladores ODBC. ODBC 3.51 incluye versiones ANSI y Unicode habilitadas para ODBC Test. Los archivos correspondientes son los siguientes:  
  
-   Odbcte32.exe y Gtrtst32.dll, para la versión ANSI.  
  
-   Odbct32w.exe y Gtrts32w.dll, para la versión Unicode.  
  
 Para usar la prueba ODBC, debe comprender la API ODBC, el lenguaje C y SQL. Para obtener más información acerca de la API ODBC, vea Referencia [del programador ODBC](../odbc/reference/odbc-programmer-s-reference.md).  
  
 Los temas de ayuda que anteriormente se incluían en esta sección de la documentación ahora se encuentran dentro del programa de prueba ODBC. Abra Odbcte32.exe u Odbct32w.exe, abra el menú **Ayuda** y, a continuación, haga clic en **Temas de ayuda**.  
  
 Tenga en cuenta que las versiones de 64 bits de estas aplicaciones, destinadas a sistemas operativos Microsoft Windows de 64 bits, tienen los mismos nombres que las versiones de 32 bits, aunque sean archivos independientes. es decir, el nombre de la versión Unicode de la versión de 64 bits de ODBC Test es odbct32w.exe.
