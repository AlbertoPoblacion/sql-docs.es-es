# <a name="translation-dll-function-summary"></a>Resumen de funciones DLL de traducción
En la tabla siguiente se describen las funciones DLL de traducción. Para obtener más información sobre la sintaxis y la semántica de cada función, vea Referencia de la [función DLL](../../../odbc/reference/syntax/translation-dll-api-reference.md)de traducción .  
  
|Tarea|Nombre de función|Propósito|  
|----------|-------------------|-------------|  
|Traducción de datos|[SQLDataSourceToDriver](../../../odbc/reference/syntax/sqldatasourcetodriver-function.md)|Traduce todos los datos que fluyen desde el origen de datos al controlador.|  
||[SQLDriverToDataSource](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)|Traduce todos los datos que fluyen desde el controlador al origen de datos.|
