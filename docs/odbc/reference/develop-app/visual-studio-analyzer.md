# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  La compatibilidad con Visual Studio Analyzer se quitó a partir de Windows 8 (Visual Studio Analyzer solo se incluía en versiones anteriores de Visual Studio.). Para un mecanismo de solución de problemas alternativo, utilice el seguimiento de BID.  
  
 Microsoft® Visual Studio™ Analyzer proporciona una vista de alto nivel de la aplicación ODBC. El desarrollador puede evaluar, analizar y depurar la estructura, el rendimiento y las interacciones desde la perspectiva de la aplicación, en lugar de desde una perspectiva de componente o código. La información recopilada por Visual Studio Analyzer está en forma de eventos, que representan algún tipo de interacción entre dos componentes de la aplicación. Los eventos se pueden componer de llamadas de función y devoluciones de funciones, como conexiones, consultas de base de datos y transacciones.
