# <a name="batch-processing"></a>Procesamiento por lotes
La compatibilidad con lotes de instrucciones SQL no está generalizada, por lo que las aplicaciones interoperables deben usarlas de forma condicional o no. Para obtener más información, vea [Ejecutar lotes](../../../odbc/reference/develop-app/executing-batches.md).
