# <a name="batch-processing"></a>Procesamiento por lotes
Compatibilidad con lotes de instrucciones SQL no está generalizada, para que aplicaciones interoperables deben usarlas condicionalmente o no admitirlo. Para obtener más información, consulte [ejecutar lotes](../../../odbc/reference/develop-app/executing-batches.md).
