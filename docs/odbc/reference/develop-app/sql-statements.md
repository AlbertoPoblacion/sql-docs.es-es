# <a name="sql-statements"></a>Instrucciones SQL
Las aplicaciones ODBC realizan casi todo el acceso a la base de datos ejecutando instrucciones SQL. La forma de estas instrucciones (codificada o construida de forma rígida en tiempo de ejecución, interoperable o específica del origen de datos, etc.) depende de las necesidades de la aplicación.
