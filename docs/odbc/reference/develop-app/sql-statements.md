# <a name="sql-statements"></a>Instrucciones SQL
Las aplicaciones ODBC realizan casi todo el acceso a la base de datos mediante la ejecución de instrucciones SQL. La forma de estas instrucciones , codificadas de forma rígida o construidas en tiempo de ejecución, interoperables o específicas del origen de datos, etc., depende de las necesidades de la aplicación.
