# <a name="cstring-class"></a>Clase CString
Dado que los objetos de la clase **CString** en Microsoft® Visual C++® están firmados y los argumentos de cadena en funciones ODBC no están firmados, las aplicaciones que pasan **CString** objetos a funciones ODBC sin convertirlos recibirán advertencias del compilador.
