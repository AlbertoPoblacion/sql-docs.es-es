# <a name="enabling-visual-studio-analyzer"></a>Habilitación de Visual Studio Analyzer
> [!IMPORTANT]  
>  La compatibilidad con Visual Studio Analyzer se quitó a partir de Windows 8 (Visual Studio Analyzer solo se incluía en versiones anteriores de Visual Studio.). Para un mecanismo de solución de problemas alternativo, utilice el seguimiento de BID.  
  
 Microsoft Visual Studio Analyzer es una herramienta de análisis de aplicaciones diseñada para proporcionar una visión de alto nivel del rendimiento de una aplicación en todos los niveles y sistemas. Se centra en la interacción entre los componentes. Visual Studio Analyzer se puede iniciar y detener desde la pestaña Seguimiento del administrador ODBC.  
  
 Para ver cualquiera de los eventos que genera ODBC, siga estos pasos:  
  
1.  Acceda al Administrador de orígenes de datos ODBC.  
  
2.  Haga clic en la pestaña **Seguimiento.**  
  
3.  Haga clic en Iniciar seguimiento de **Visual Studio Analyzer**.  
  
4.  Haga clic en **OK**.  
  
 La generación de eventos de Visual Studio Analyzer continúa hasta que se selecciona **Detener seguimiento de Visual Studio Analyzer.**
