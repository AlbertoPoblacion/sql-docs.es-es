# <a name="enabling-visual-studio-analyzer"></a>Habilitación de Visual Studio Analyzer
> [!IMPORTANT]  
>  Se quitó la compatibilidad con Visual Studio Analyzer a partir de Windows 8 (Visual Studio Analyzer solo se incluía en versiones anteriores de Visual Studio). Para un mecanismo de solución de problemas alternativo, use el seguimiento de PUJAs.  
  
 Microsoft Visual Studio Analyzer es una herramienta de análisis de aplicaciones diseñada para proporcionar una visión de alto nivel del rendimiento de una aplicación en todos los niveles y sistemas. Se centra en la interacción entre los componentes. Visual Studio Analyzer puede iniciarse y detenerse en la pestaña seguimiento del administrador de ODBC.  
  
 Para ver cualquiera de los eventos generados por ODBC, siga estos pasos:  
  
1.  Obtenga acceso al administrador de orígenes de datos ODBC.  
  
2.  Haga clic en la pestaña **seguimiento** .  
  
3.  Haga clic en **iniciar seguimiento de Visual Studio Analyzer**.  
  
4.  Haga clic en **OK**.  
  
 Visual Studio Analyzer la generación de eventos continúa hasta que se selecciona **detener Visual Studio Analyzer seguimiento** .
