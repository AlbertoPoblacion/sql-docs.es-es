# <a name="parameter-markers-in-procedure-calls"></a>Marcadores de parámetros en las llamadas a procedimiento
Al llamar a procedimientos que aceptan parámetros, las aplicaciones interoperables deben usar marcadores de parámetros en lugar de valores de parámetro literales. Algunos orígenes de datos no admiten el uso de valores de parámetros literales en las llamadas a procedimientos. Para obtener más información acerca de los parámetros, vea Parámetros de [instrucción](../../../odbc/reference/develop-app/statement-parameters.md). Para obtener más información acerca de los procedimientos de llamada, vea [llamadas](../../../odbc/reference/develop-app/procedure-calls.md)a procedimientos , más adelante en esta sección.
