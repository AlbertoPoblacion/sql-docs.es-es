# <a name="backward-compatibility-and-standards-compliance"></a>Compatibilidad con versiones anteriores y el cumplimiento de estándares
La compatibilidad con versiones anteriores es la capacidad de los componentes ODBC más recientes para trabajar con componentes ODBC antiguos. En las secciones siguientes se describe cómo estos componentes se ven afectados por los cambios en ODBC *3.x*. La información contenida en ellos aborda principalmente la escritura de una aplicación ODBC *3.x* y cómo los problemas de compatibilidad con versiones anteriores se controlan mediante controladores ODBC. Para obtener instrucciones específicas sobre cómo afectan los problemas de compatibilidad con versiones anteriores a la escritura de un controlador ODBC *3.x,* vea [Apéndice G: Directrices](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md)de controlador para la compatibilidad con versiones anteriores .
