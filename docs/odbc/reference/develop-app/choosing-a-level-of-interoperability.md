# <a name="choosing-a-level-of-interoperability"></a>Elección de un nivel de interoperabilidad
Suponiendo que la aplicación va a utilizar ODBC, el siguiente paso es determinar qué nivel de interoperabilidad se requiere. El nivel básico de interoperabilidad suele ser una función del tipo de aplicación: las aplicaciones personalizadas tienden a no ser interoperables, las aplicaciones verticales tienden a interoperar entre un número limitado de DBMS y las aplicaciones genéricas tienden a interoperar entre todos los DBMS.
