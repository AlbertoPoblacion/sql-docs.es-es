# <a name="choosing-a-level-of-interoperability"></a>Elección de un nivel de interoperabilidad
Suponiendo que la aplicación usará ODBC, el siguiente paso es determinar qué nivel de interoperabilidad es necesario. El nivel básico de interoperabilidad suele ser una función del tipo de aplicación: las aplicaciones personalizadas tienden a no ser interoperables, las aplicaciones verticales tienden a ser interoperables entre un número limitado de DBMS y las aplicaciones genéricas tienden a ser interoperables entre todos los DBMS.
