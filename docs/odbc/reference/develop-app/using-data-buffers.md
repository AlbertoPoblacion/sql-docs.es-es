# <a name="using-data-buffers"></a>Utilizar los búferes de datos
Los búferes de datos se describen mediante tres elementos de información: su tipo, dirección y longitud de bytes. Siempre que una función necesita uno de estos elementos de información y aún no lo conoce, tiene un argumento con el que la aplicación la pasa.  
  
 Esta sección contiene los temas siguientes.

-   [Dirección del búfer de datos](../../../odbc/reference/develop-app/data-buffer-address.md)  
  
-   [Longitud del búfer de datos](../../../odbc/reference/develop-app/data-buffer-length.md)
