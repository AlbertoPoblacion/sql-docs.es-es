# <a name="pseudo-type-identifiers"></a>Identificadores de pseudotipo
Para la comodidad de programación de aplicaciones, ODBC define un número de identificadores de pseudotipo. Estos identificadores no se corresponden realmente con los tipos de datos reales, sino que, en su lugar, dependiendo de la situación, se resuelven en los tipos de datos existentes.  
  
 Esta sección contiene los temas siguientes.

-   [SQL_ARD_TYPE](../../../odbc/reference/appendixes/sql-ard-type.md)  
  
-   [SQL_C_TCHAR](../../../odbc/reference/appendixes/sql-c-tchar.md)
