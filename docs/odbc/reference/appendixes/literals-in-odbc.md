# <a name="literals-in-odbc"></a>Literales de ODBC
La sintaxis de las secciones siguientes se utiliza para los literales numéricos y de intervalo en ODBC. Esta sintaxis se proporciona aquí como una ayuda para los escritores de controladores cuando se realizan conversiones de un tipo de cadena de caracteres a un tipo numérico o de intervalo, o de un tipo numérico o de intervalo a un tipo de cadena de caracteres. Para obtener más información, consulte [Literales](../../../odbc/reference/appendixes/interval-literals.md) de intervalo y [literales numéricos](../../../odbc/reference/appendixes/numeric-literals.md) en Apéndice D: tipos de datos.  
  
 Este apéndice contiene los siguientes temas.  
  
-   [Sintaxis de literales de intervalo](../../../odbc/reference/appendixes/interval-literal-syntax.md)  
  
-   [Sintaxis de los literales numéricos](../../../odbc/reference/appendixes/numeric-literal-syntax.md)
