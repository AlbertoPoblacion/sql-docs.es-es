# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations y biblioteca de cursores
> [!IMPORTANT]  
>  Esta característica se quitará en una versión futura de Windows. Evite usar esta característica en los nuevos trabajos de desarrollo y planee modificar las aplicaciones que actualmente la utilizan. Microsoft recomienda el uso de la funcionalidad de cursor del controlador.  
  
 La biblioteca de cursores no admite la llamada a **SQLBulkOperations**. Para obtener información general sobre **SQLBulkOperations**, consulte la [función SQLBulkOperations](../../../odbc/reference/syntax/sqlbulkoperations-function.md).
