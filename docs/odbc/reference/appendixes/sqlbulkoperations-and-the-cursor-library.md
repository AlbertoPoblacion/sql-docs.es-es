# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations y biblioteca de cursores
> [!IMPORTANT]  
>  Esta característica se eliminará en una versión futura de Windows. Evite usar esta característica en el nuevo trabajo de desarrollo y planee modificar las aplicaciones que actualmente utilizan esta característica. Microsoft recomienda usar la funcionalidad del cursor del controlador.  
  
 La biblioteca de cursores no admite la llamada a **SQLBulkOperations**. Para obtener información general acerca de **SQLBulkOperations**, vea [SQLBulkOperations (función).](../../../odbc/reference/syntax/sqlbulkoperations-function.md)
