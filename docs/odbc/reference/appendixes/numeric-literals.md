# <a name="numeric-literals"></a>Literales numéricos
Los literales numéricos se utilizan cuando los valores de datos numéricos se almacenan en cadenas de caracteres. Para la conversión de datos SQL numéricos a una cadena de SQL_C_CHAR o la conversión de datos numéricos de C a una cadena SQL_CHAR o SQL_VARCHAR, se usa la sintaxis literal numérica para especificar lo que se almacena en el destino. Para la conversión de un número almacenado como una cadena de SQL_C_CHAR a datos SQL numéricos, o un número almacenado como una cadena SQL_CHAR a datos numéricos de C, esta sintaxis se usa para validar lo que se almacena en el origen.  
  
 Los literales numéricos deben ajustarse a la sintaxis definida en la sección [Sintaxis literal numérica](../../../odbc/reference/appendixes/numeric-literal-syntax.md) en apéndice C: gramática SQL.
