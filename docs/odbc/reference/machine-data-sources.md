# <a name="machine-data-sources"></a>Orígenes de datos de equipo
*Orígenes de datos de equipo* se almacenan en el sistema con un nombre definido por el usuario. Toda la información asociada con el nombre del origen de datos es la necesidad de administrador de controladores y el controlador para conectarse al origen de datos. Para un origen de datos de Xbase, podría tratarse del nombre del controlador Xbase, la ruta de acceso completa del directorio que contiene los archivos de Xbase y algunas opciones que indican al controlador de cómo usar esos archivos, como el modo de usuario único o de solo lectura. Para un origen de datos de Oracle, esto podría ser el nombre del controlador Oracle, el servidor donde reside el DBMS de Oracle, SQL * Net de cadena de conexión que identifica el SQL\*controlador de red para usar y el identificador de la base de datos en el servidor de sistema.
