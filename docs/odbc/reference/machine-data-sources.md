# <a name="machine-data-sources"></a>Orígenes de datos de equipo
*Los orígenes* de datos de la máquina se almacenan en el sistema con un nombre definido por el usuario. Asociado con el nombre del origen de datos es toda la información que el Administrador de controladores y el controlador necesitan para conectarse al origen de datos. Para un origen de datos Xbase, este podría ser el nombre del controlador Xbase, la ruta de acceso completa del directorio que contiene los archivos Xbase y algunas opciones que indican al controlador cómo utilizar esos archivos, como el modo de usuario único o de solo lectura. Para un origen de datos de Oracle, este podría ser el nombre del controlador de Oracle, el\*servidor donde reside el DBMS de Oracle, la cadena de conexión SQL*Net que identifica el controlador de SQL Net que se va a utilizar y el identificador del sistema de la base de datos en el servidor.
