# <a name="types-of-data-sources"></a>Tipos de orígenes de datos
Hay dos tipos de orígenes de datos: orígenes de datos de máquina y orígenes de datos de archivo. Aunque ambos contienen información similar sobre el origen de los datos, difieren en la forma en que se almacena esta información. Debido a estas diferencias, se utilizan de maneras algo diferentes.  
  
 Esta sección contiene los temas siguientes.  
  
-   [Orígenes de datos de equipo](../../odbc/reference/machine-data-sources.md)  
  
-   [Orígenes de datos de archivo](../../odbc/reference/file-data-sources.md)
