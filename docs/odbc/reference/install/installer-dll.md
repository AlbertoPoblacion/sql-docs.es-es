# <a name="installer-dll"></a>DLL instalador
> [!NOTE]  
>  A partir de Windows XP y Windows Server 2003, ODBC se incluye en el sistema operativo Windows. Solo debe instalar ODBC explícitamente en versiones anteriores de Windows.  
  
 El archivo DLL del instalador contiene funciones para instalar y quitar componentes ODBC, mantener información del Registro sobre esos componentes y mantener información del Registro sobre los orígenes de datos. Está escrito por Microsoft y se puede redistribuir. Para obtener una descripción completa de las funciones del archivo DLL del instalador, consulte Referencia de la API de [DLL del instalador](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
