# <a name="default-subkey"></a>Subclave predeterminada
El registro puede especificar un origen de datos predeterminado con la subclave Default. Esta subclave es un caso especial de una subclave de especificación de origen de datos y tiene los mismos valores que cualquier otra subclave de especificación de origen de datos. La única diferencia es que no aparece como un valor en la subclave Orígenes de datos ODBC.
