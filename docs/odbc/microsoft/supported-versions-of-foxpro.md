# <a name="supported-versions-of-foxpro"></a>Versiones compatibles de FoxPro
Puede usar el controlador ODBC de Microsoft® Visual FoxPro® para tener acceso a los datos almacenados en las tablas de FoxPro. Se admiten las siguientes versiones de datos de FoxPro:  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   Visual FoxPro (todas las versiones)  
  
 Al acceder a los datos almacenados en Visual FoxPro, puede optar por conectarse a una base de [datos](../../odbc/microsoft/visual-foxpro-terminology.md) que contenga cero o más [tablas](../../odbc/microsoft/visual-foxpro-terminology.md) o a un directorio de [tablas libres.](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Para obtener más información acerca de cómo conectarse a un origen de datos, vea Agregar un origen de datos de [Visual FoxPro](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md).  
  
 Para descargar el controlador, consulte el sitio de descarga del controlador ODBC de [Visual FoxPro](https://go.microsoft.com/fwlink/?LinkId=121318).
