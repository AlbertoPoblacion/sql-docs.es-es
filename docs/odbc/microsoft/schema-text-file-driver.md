# <a name="schema-text-file-driver"></a>Esquema (controlador de archivo de texto)
El esquema incluye información sobre cada tabla (archivo de texto) en un origen de datos, incluido el formato de la tabla, el número de filas que se van a examinar para determinar los tipos de columna, si la primera fila de la tabla contiene nombres de columna, si el archivo de código fuente se escribe con una página de códigos OEM o ANSI, y el nombre, tipo de datos y ancho
