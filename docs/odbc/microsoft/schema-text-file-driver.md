# <a name="schema-text-file-driver"></a>Esquema (controlador de archivo de texto)
El esquema incluye información acerca de cada tabla (archivo de texto) en un origen de datos, incluido el formato de la tabla, el número de filas que se va a examinar para determinar los tipos de columna, si la primera fila de la tabla contiene los nombres de columna, si se escribe el archivo de origen utilizando un OEM o Página de códigos ANSI y nombre de cada columna, tipo de datos y ancho.
