# <a name="files-to-use-with-the-desktop-database-drivers"></a>Archivos que se utilizan con los controladores de escritorio de la base de datos
Los controladores de base de datos de escritorio de ODBC se utilizan con el motor de base de datos Microsoft Jet, archivos ISAM instalables y archivos de Microsoft ODBC. No todos son necesarios para la redistribución del producto, y no todos son los componentes principales.  
  
|Para obtener información acerca de|Vea|  
|---------------------------|---------|  
|Archivos utilizados por cada controlador de base de datos de escritorio|Archivos del controlador ODBC|  
|Archivos de Microsoft Jet utilizados por cada controlador de base de datos de escritorio|Archivos de Microsoft Jet|  
|Archivos de controlador IISAM que proporcionan acceso a los datos entre Microsoft Jet y el otro DBMS|Archivos de controlador ISAM instalables|
