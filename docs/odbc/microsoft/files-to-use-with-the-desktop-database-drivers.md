# <a name="files-to-use-with-the-desktop-database-drivers"></a>Archivos que se utilizan con los controladores de escritorio de la base de datos
Los controladores de base de datos de escritorio ODBC se utilizan con Microsoft Jet Motor de base de datos, los archivos ISAM instalables y los archivos ODBC de Microsoft. No todos son necesarios para la redistribución del producto y no todos son componentes principales.  
  
|Para información acerca de|Vea|  
|---------------------------|---------|  
|Archivos usados por cada controlador de base de datos de escritorio|Archivos del controlador ODBC|  
|Archivos de Microsoft Jet usados por cada controlador de base de datos de escritorio|Archivos de Microsoft Jet|  
|Archivos de controlador IISAM que proporcionan acceso a datos entre Microsoft Jet y el otro DBMS|Archivos de controlador ISAM instalables|
