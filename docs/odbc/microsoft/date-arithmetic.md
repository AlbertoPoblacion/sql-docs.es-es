# <a name="date-arithmetic"></a>Aritmética de fecha
El controlador admite agregar y restar un entero de una columna DATE, TIME o TIMESTAMP. Para una columna DATE, el entero especifica el número de días que se agregará o restará. Para una columna TIME o TIMESTAMP, el entero especifica el número de segundos que se deben agregar o restar.
