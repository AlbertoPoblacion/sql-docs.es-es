# <a name="date-arithmetic"></a>Aritmética de fecha
El controlador permite agregar y restar un entero de una columna de fecha, hora o marca de tiempo. En el caso de una columna de fecha, el entero especifica el número de días que se van a sumar o restar. En el caso de una columna de hora o marca de tiempo, el entero especifica el número de segundos que se van a sumar o restar.
