# <a name="sqlnumresultcols-visual-foxpro-odbc-driver"></a>SQLNumResultCols (controlador ODBC de Visual FoxPro)
> [!NOTE]

 Compatibilidad: completa

 Devuelve el número de columnas de un cursor de conjunto de resultados.  
  
 Para obtener más información, vea [SQLNumResultCols](../../odbc/reference/syntax/sqlnumresultcols-function.md) en la *Referencia del programador de ODBC*.
