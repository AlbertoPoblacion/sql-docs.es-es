# <a name="additional-supported-odbc-sql-grammar"></a>Gramática de SQL de ODBC compatibles adicionales
Los controladores ODBC admiten completamente la gramática mínima ODBC 2.5 y las cláusulas siguientes en la gramática básicos y extendidos 2.5 de ODBC. Se puede mostrar información adicional yendo a la [instrucción ALTER TABLE](../../odbc/microsoft/alter-table-statement.md), [nombres de correlación](../../odbc/microsoft/correlation-names.md), y [invocación del procedimiento](../../odbc/microsoft/procedure-invocation.md) temas.  
  
-   Literal numérico aproximado (núcleos)  
  
-   [Instrucción ALTER TABLE](../../odbc/microsoft/alter-table-statement.md) (núcleos)  
  
-   Literales binarios (extendido)  
  
-   [Nombres de correlación](../../odbc/microsoft/correlation-names.md) (núcleos)  
  
-   [Invocación del procedimiento](../../odbc/microsoft/procedure-invocation.md) (extendido)  
  
-   [ENTRE el predicado](../../odbc/microsoft/between-predicate.md) (núcleos)  
  
-   [Instrucción CREATE INDEX](../../odbc/microsoft/create-index-statement.md)  
  
-   [Aritmética de fecha](../../odbc/microsoft/date-arithmetic.md)  
  
-   [Jet: Fecha, hora y marca de tiempo literales](../../odbc/microsoft/jet-date-time-and-timestamp-literals.md)  
  
-   [Instrucción DROP INDEX](../../odbc/microsoft/drop-index-statement.md)  
  
-   [Archivo de texto de ancho fijo](../../odbc/microsoft/fixed-width-text-file.md)  
  
-   [GROUP BY-lista de expresiones](../../odbc/microsoft/group-by-expression-list.md)  
  
-   [ORDER BY-lista de expresiones](../../odbc/microsoft/order-by-expression-list.md)  
  
-   [ORDER BY con GROUP BY](../../odbc/microsoft/order-by-with-group-by.md)  
  
-   Predicado EXISTS (núcleos)  
  
-   IN (*valuelist*) (núcleos)  
  
-   [Jet: Combinaciones externas](../../odbc/microsoft/jet-outer-joins.md) (extendido)  
  
-   Instrucción SELECT (extendido) (incluido GROUP BY, HAVING y cláusulas de unión)  
  
-   Subconsultas (subselecciones) (núcleos)  
  
-   [Funciones escalares](../../odbc/microsoft/scalar-functions.md)  
  
-   [Nombres de tabla](../../odbc/microsoft/table-names.md)
