# <a name="odbc-component-files"></a>Archivos de componentes ODBC
Cada componente de base de datos de escritorio de ODBC Drivers 4.0 requiere los archivos del controlador ODBC.  
  
 Algunos archivos se requiere más de un componente. Estos archivos deben distribuirse, si tiene previsto redistribuir cualquiera de los controladores de base de datos de escritorio de ODBC con las aplicaciones comerciales.  
  
 Estos archivos se instalan en el directorio \Windows\System para Windows 95 o versiones posteriores, o en el directorio de \Windows\System32 para Windows NT 3.51 o versiones posteriores y Windows 2000. Si los archivos ODBC se instalaron anteriormente en un directorio diferente, asegúrese de que usa el directorio de archivos en \Windows\System (o System32) más reciente.
