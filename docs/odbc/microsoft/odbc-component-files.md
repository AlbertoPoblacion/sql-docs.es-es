# <a name="odbc-component-files"></a>Archivos de componentes ODBC
Los archivos del controlador ODBC son necesarios para cada componente de los controladores de base de datos de escritorio ODBC 4,0.  
  
 Algunos archivos son necesarios para más de un componente. Estos archivos deben redistribuirse si tiene previsto redistribuir cualquiera de los controladores de base de datos de escritorio ODBC con las aplicaciones comerciales.  
  
 Estos archivos se instalan en el directorio \Windows\System para Windows 95 o posterior, o en el directorio \windows\system32. para Windows NT 3,51 o posterior y Windows 2000. Si los archivos ODBC se instalaron previamente en un directorio diferente, asegúrese de usar los archivos más recientes en el directorio \Windows\System (o system32).
