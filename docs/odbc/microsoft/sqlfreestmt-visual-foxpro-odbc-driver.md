# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt (controlador ODBC de Visual FoxPro)
> [!NOTE]  
>  Este tema contiene información específica del controlador ODBC de Visual FoxPro. Para obtener información general sobre esta función, consulte el tema adecuado en Referencia de [la API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Soporte: Completo  
  
 Conformidad de la API ODBC: nivel principal  
  
 Detiene el procesamiento asociado a un *hstmt*específico, cierra los cursores abiertos asociados con *hstmt*, descarta los resultados pendientes y, opcionalmente, libera todos los recursos asociados con el identificador de instrucción.  
  
 Para obtener más información, vea [SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) en la *referencia del programador ODBC*.
