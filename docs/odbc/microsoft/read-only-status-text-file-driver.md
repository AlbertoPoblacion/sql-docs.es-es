# <a name="read-only-status-text-file-driver"></a>Estado de solo lectura (controlador de archivo de texto)
Cuando se utiliza el controlador de texto, las tablas de origen de datos son de solo lectura. Sin embargo, las aplicaciones pueden realizar inserciones para tablas de texto.
