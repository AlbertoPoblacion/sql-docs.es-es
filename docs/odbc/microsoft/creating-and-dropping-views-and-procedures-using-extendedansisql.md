# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Creación y eliminación de vistas y procedimientos con ExtendedAnsiSQL
Cuando la marca ExtendedAnsiSQL está activada, jet 4,0 admite la capacidad de crear y quitar vistas y procedimientos a través de SQL. Estas vistas y procedimientos no estarán visibles a través del acceso.
