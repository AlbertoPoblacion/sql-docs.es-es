# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>Crear etiquetas postales en Microsoft Word usando datos de Visual FoxPro
Puede usar los datos de Visual FoxPro en un documento de Microsoft Word para Windows 95 o Windows 98. Por ejemplo, puede que desee crear etiquetas de correo a partir de la información del cliente almacenada en una tabla de Visual FoxPro.  
  
### <a name="to-create-mailing-labels"></a>Para crear etiquetas de correo  
  
1.  En Microsoft Word, cree un nuevo documento en blanco.  
  
2.  En el menú herramientas, elija combinación de correspondencia.  
  
3.  En la aplicación auxiliar de combinación de correspondencia, elija crear y, a continuación, seleccione Etiquetas de correo.  
  
4.  En documento principal, elija ventana activa.  
  
5.  En origen de datos, elija obtener datos y, a continuación, seleccione Abrir origen de datos.  
  
6.  En el cuadro de diálogo Abrir origen de datos, elija MS Query.  
  
7.  En el cuadro de diálogo Seleccionar origen de datos, seleccione un origen de datos de Visual FoxPro y, a continuación, haga clic en usar.  
  
8.  Si la base de datos a la que tiene acceso el origen de datos incluye tablas, seleccione una tabla en el cuadro de diálogo Agregar tablas. Microsoft Query muestra la tabla agregada en la mitad superior del diseñador de consultas.  
  
9. Seleccione los campos de la consulta arrastrándolos desde la tabla hasta la mitad inferior del diseñador.  
  
10. En el menú Archivo, elija devolver datos a Microsoft Word. Microsoft Query se cierra y los datos seleccionados están disponibles para su uso en el documento de combinación de correspondencia.  
  
11. En documento principal, elija Configuración.  
  
12. En el cuadro de diálogo Opciones de etiqueta, seleccione la información de la impresora y la etiqueta que desee y, a continuación, haga clic en Aceptar.  
  
13. En el cuadro de diálogo crear etiquetas, seleccione los campos que desea imprimir en las etiquetas de correo y, a continuación, haga clic en Aceptar.  
  
14. En la aplicación auxiliar de combinación de correspondencia, en combinar los datos con el documento, haga clic en combinar.  
  
15. En el cuadro de diálogo combinar, seleccione las opciones que desee y, a continuación, haga clic en combinar.
