# <a name="visual-foxpro-odbc-driver"></a>Controlador ODBC de Visual FoxPro
Microsoft Visual FoxPro es un potente entorno orientado a objetos para la construcción de bases de datos y el desarrollo de aplicaciones. El controlador ODBC de Microsoft Visual FoxPro permite a las aplicaciones abrir, consultar y actualizar datos en Visual FoxPro y versiones anteriores de FoxPro a través de la interfaz Open Database Connectivity (ODBC).  
  
 Por ejemplo, con el controlador ODBC de Microsoft Visual FoxPro puede hacer lo siguiente:  
  
-   Use Microsoft Query para consultar y actualizar datos de Visual FoxPro desde hojas de cálculo de Microsoft Excel.  
  
-   Cree cartas de combinación de correspondencia con datos de Visual FoxPro con Microsoft Word.  
  
-   Consultar y actualizar vistas y tablas de Visual FoxPro desde Microsoft Access.  
  
-   Use Visual FoxPro como almacén de datos para aplicaciones Microsoft Visual Basic, Microsoft Visual C++ y C.
