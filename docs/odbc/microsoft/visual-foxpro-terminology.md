# <a name="visual-foxpro-terminology"></a>Terminología de Visual FoxPro
**base**  
 En Visual FoxPro, un archivo de base de datos tiene una extensión. DBC y puede contener una o más **tablas**.  
  
 **tabla de base de datos**  
 En Visual FoxPro, una tabla que está asociada a una base de datos. **Tabla libre**de contraste.  
  
 **tabla libre**  
 En Visual FoxPro, una tabla que no está asociada a una base de datos.  
  
 Un archivo. DBF creado en la versión 2. x de FoxPro es una tabla libre a menos que se convierta en una tabla de Visual FoxPro y se agregue a una base de datos de Visual FoxPro. **Tabla de base de datos**de contraste.  
  
 **instrucción SQL preparable.**  
 Instrucción SQL que no ha sido procesada ya por la función **SQLPrepare** . Para obtener más información acerca de esta función con el controlador ODBC de Visual FoxPro, consulte [SQLPrepare (controlador ODBC de Visual FoxPro)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 **cuadro**  
 En Visual FoxPro, los registros se almacenan en una tabla. Cada fila de una tabla representa un registro y las columnas de la tabla representan los campos del registro. Cada tabla de Visual FoxPro se almacena en su propio archivo con la extensión. DBF. Las tablas de Visual FoxPro se pueden asociar a una base de datos.  
  
 Versión 2 de FoxPro. las tablas *x* no están asociadas a una base de datos.
