# <a name="visual-foxpro-terminology"></a>Terminología de Visual FoxPro
**database**  
 En Visual FoxPro, un archivo de base de datos tiene una extensión .dbc y puede contener uno o varios **tablas**.  
  
 **tabla de base de datos**  
 En Visual FoxPro, una tabla que está asociado con una base de datos. Contraste **tabla libre**.  
  
 **tabla libre**  
 En Visual FoxPro, una tabla que no está asociada a una base de datos.  
  
 Creado un archivo .dbf en FoxPro versión 2.x es una tabla libre a menos que se convierte en una tabla de Visual FoxPro y agregado a una base de datos de Visual FoxPro. Contraste **tabla de base de datos**.  
  
 **instrucción SQL preparable.**  
 Una instrucción SQL que ya no se ha procesado por el **SQLPrepare** función. Para obtener más información acerca de esta función con el controlador ODBC de Visual FoxPro, consulte [SQLPrepare (controlador de ODBC de Visual FoxPro)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 **table**  
 En Visual FoxPro, los registros se almacenan en una tabla. Cada fila de una tabla representa un registro y las columnas de la tabla representan los campos del registro. Cada tabla de Visual FoxPro se almacena en su propio archivo con una extensión .dbf. Tablas de Visual FoxPro pueden asociarse con una base de datos.  
  
 FoxPro versión 2. *x* tablas no están asociadas con una base de datos.
