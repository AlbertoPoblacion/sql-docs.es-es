# <a name="visual-foxpro-terminology"></a>Terminología de Visual FoxPro
**Base**  
 En Visual FoxPro, un archivo de base de datos tiene una extensión .dbc y puede contener una o varias **tablas.**  
  
 **tabla de base de datos**  
 En Visual FoxPro, una tabla asociada a una base de datos. Mesa **libre de**contraste .  
  
 **mesa libre**  
 En Visual FoxPro, una tabla que no está asociada a una base de datos.  
  
 Un archivo .dbf creado en FoxPro versión 2.x es una tabla gratuita a menos que se convierta en una tabla de Visual FoxPro y se agregue a una base de datos de Visual FoxPro. Tabla de **base de datos**de contraste .  
  
 **declaración SQL preparable**  
 Una instrucción SQL que aún no ha sido procesada por la función **SQLPrepare.** Para obtener más información acerca de esta función con el controlador ODBC de Visual FoxPro, vea [SQLPrepare (controlador ODBC de Visual FoxPro)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 **table**  
 En Visual FoxPro, los registros se almacenan en una tabla. Cada fila de una tabla representa un registro y las columnas de la tabla representan los campos del registro. Cada tabla de Visual FoxPro se almacena en su propio archivo con la extensión .dbf. Las tablas de Visual FoxPro se pueden asociar a una base de datos.  
  
 FoxPro versión 2. *x* tablas no están asociadas a una base de datos.
