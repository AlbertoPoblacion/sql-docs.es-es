# <a name="order-by-clause-limitations"></a>ORDEN por cláusula limitaciones
Si una instrucción SELECT contiene una cláusula GROUP BY y una cláusula ORDER BY, la cláusula ORDER BY solo puede contener una columna del conjunto de resultados o una expresión de la cláusula GROUP BY.
