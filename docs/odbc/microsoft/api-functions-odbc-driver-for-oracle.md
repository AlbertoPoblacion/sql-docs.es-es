# <a name="api-functions-odbc-driver-for-oracle"></a>Funciones de la API (controlador ODBC para Oracle)
> [!IMPORTANT]  
>  Esta característica se eliminará en una versión futura de Windows. Evite utilizar esta característica en nuevos trabajos de desarrollo y tenga previsto modificar las aplicaciones que actualmente la utilizan. En su lugar, utilice el controlador ODBC proporcionado por Oracle.  
  
 El controlador ODBC de Microsoft para Oracle admite las funciones de API de nivel principal, nivel 1 y nivel 2. Estas funciones se enumeran en Niveles de [conformidad ODBC](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md).  
  
-   La conformidad de [la interfaz](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) de nivel de núcleo (CLI) proporciona las características definidas en la especificación ISO CLI y las características obligatorias definidas en la especificación CLI del grupo abierto.  
  
-   [La](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) conformidad de nivel 1 proporciona funcionalidad de interfaz de nivel básico, así como características adicionales, como transacciones.  
  
-   [La](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) conformidad de nivel 2 proporciona funcionalidad de nivel 1, así como características adicionales como marcadores, parámetros dinámicos y ejecución asincrónica de funciones ODBC.
