# <a name="api-functions-odbc-driver-for-oracle"></a>Funciones de la API (controlador ODBC para Oracle)
> [!IMPORTANT]  
>  Esta característica se quitará en una versión futura de Windows. Evite utilizar esta característica en nuevos trabajos de desarrollo y tenga previsto modificar las aplicaciones que actualmente la utilizan. En su lugar, utilice el controlador ODBC proporcionado por Oracle.  
  
 Microsoft ODBC driver for Oracle admite las funciones de API de nivel básico, nivel 1 y nivel 2. Estas funciones se enumeran en [los niveles de cumplimiento de ODBC](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md).  
  
-   La conformidad con la [interfaz de nivel básico](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) (CLI) proporciona características definidas en la especificación de la CLI ISO y las características obligatorias que se definen en la especificación Open Group CLI.  
  
-   El cumplimiento de [nivel 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) proporciona funcionalidad de interfaz de nivel básico, así como características adicionales, como transacciones.  
  
-   El cumplimiento de [nivel 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) proporciona funcionalidad de nivel 1, así como características adicionales, como marcadores, parámetros dinámicos y la ejecución asincrónica de funciones ODBC.
