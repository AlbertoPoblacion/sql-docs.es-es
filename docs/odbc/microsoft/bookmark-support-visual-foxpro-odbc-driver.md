# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>Compatibilidad con marcadores (controlador ODBC de Visual FoxPro)
El controlador ODBC de Visual FoxPro admite marcadores simples. Cuando se llama a [SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md) con el SQL_BOOKMARK_PERSISTENCE *InfoType*, el valor devuelto se SQL_BP_SCROLL.  
  
 Para obtener más información acerca de los marcadores, vea [Marcadores (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md).
