# <a name="alter-table-statement"></a>Instrucción ALTER TABLE
Cuando se usa Microsoft Access, dBASE o Paradoxdriver, se admite la instrucción ALTER TABLE con las cláusulas Add y DROP TABLE Constraint Definition. (No se admiten las instrucciones ALTER TABLE para los controladores de texto o de Microsoft Excel).  
  
> [!NOTE]  
>  Cuando se usa el controlador de Paradox sin implementar el Motor de base de datos de Borland, no se admiten las instrucciones ALTER TABLE. solo se permiten las instrucciones Read y Append.
