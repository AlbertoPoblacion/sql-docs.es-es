# <a name="sqlnumparams-visual-foxpro-odbc-driver"></a>SQLNumParams (controlador ODBC de Visual FoxPro)
> [!NOTE]  
>  Este tema contiene información específica del controlador ODBC de Visual FoxPro. Para obtener información general sobre esta función, consulte el tema adecuado en Referencia de [la API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Soporte: Completo  
  
 Conformidad de la API ODBC: Nivel 2  
  
 Devuelve el número de parámetros de una instrucción SQL. El número de parámetros debe ser igual al número de signos de interrogación de la instrucción SQL pasada a [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 Para obtener más información acerca de la gramática SQL, vea [Gramática SQL ODBC admitida](../../odbc/microsoft/supported-odbc-sql-grammar-visual-foxpro-odbc-driver.md). Para obtener más información acerca de esta función, vea [SQLNumParams](../../odbc/reference/syntax/sqlnumparams-function.md) en la *referencia del programador ODBC*.
