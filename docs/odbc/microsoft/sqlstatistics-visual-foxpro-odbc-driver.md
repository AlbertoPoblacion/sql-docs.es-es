# <a name="sqlstatistics-visual-foxpro-odbc-driver"></a>SQLStatistics (controlador ODBC de Visual FoxPro)
> [!NOTE]  
>  Este tema contiene información específica del controlador ODBC de Visual FoxPro. Para obtener información general sobre esta función, consulte el tema adecuado en Referencia de [la API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Soporte: Completo  
  
 Conformidad de la API ODBC: Nivel 1  
  
 Recupera una lista de estadísticas sobre una sola tabla y los índices, o nombres de etiqueta, asociados a la tabla. El controlador devuelve la información como un conjunto de resultados.  
  
 Para obtener más información, vea [SQLStatistics](../../odbc/reference/syntax/sqlstatistics-function.md) en la *referencia del programador ODBC*.
