# <a name="sqlputdata-visual-foxpro-odbc-driver"></a>SQLPutData (controlador ODBC de Visual FoxPro)
> [!NOTE]  
>  Este tema contiene información específica del controlador ODBC de Visual FoxPro. Para obtener información general acerca de esta función, vea el tema correspondiente en [referencia de la API de ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Soporte técnico: Completo  
  
 Conformidad de la API de ODBC: Nivel 1  
  
 Permite que una aplicación enviar datos de un parámetro o columna a del controlador en el tiempo de ejecución de la instrucción.  
  
 Para obtener más información, consulte [SQLPutData](../../odbc/reference/syntax/sqlputdata-function.md) en el *referencia del programador de ODBC*.
