# <a name="sqlcolumns-visual-foxpro-odbc-driver"></a>SQLColumns (controlador ODBC de Visual FoxPro)
> [!NOTE]  
>  Este tema contiene información específica del controlador ODBC de Visual FoxPro. Para obtener información general sobre esta función, consulte el tema adecuado en Referencia de [la API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Soporte: Completo  
  
 Conformidad de la API ODBC: Nivel 1  
  
 Para una tabla, crea un conjunto de resultados que es la lista de columnas para la tabla o tablas especificadas.  
  
 Para obtener más información, vea [SQLColumns](../../odbc/reference/syntax/sqlcolumns-function.md) en la *referencia del programador ODBC*.
