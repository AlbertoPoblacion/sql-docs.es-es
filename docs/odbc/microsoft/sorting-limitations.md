# <a name="sorting-limitations"></a>Limitaciones de ordenación
La longitud máxima de un criterio de ordenación en una cláusula GROUP BY, la cláusula ORDER BY, la instrucción SELECT DISTINCT o la combinación externa es 255 bytes; la longitud máxima de todas las claves de ordenación en una fila de la ordenación es 65.500 bytes.
