# <a name="product-support"></a>Servicios de soporte técnico
Los servicios de soporte técnico de Microsoft proporcionan soporte técnico de productos para ODBC. Dado que muchos productos de Microsoft usan ODBC como componente principal y también redistribuyen controladores ODBC, Servicios de soporte técnico de Microsoft considera que ODBC forma parte de cada producto que incorpora ODBC como componente. El soporte para ODBC se proporciona de acuerdo con el acuerdo de soporte de ese producto. Consulte la documentación del producto individual para obtener más información.
