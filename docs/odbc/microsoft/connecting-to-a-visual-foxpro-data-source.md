Puede conectarse a un origen de datos de Visual FoxPro mediante la aplicación de Microsoft Office o mediante la API de SQL.  
  
|Para conectarse desde|Vea|  
|---------------------|---------|  
|Microsoft Access, Microsoft Excel o Microsoft Word|[Acceso a datos de Visual FoxPro desde Microsoft Office](../../odbc/microsoft/accessing-visual-foxpro-data-from-microsoft-office.md)|  
|Su aplicación C o C++|[SQLConnect](../../odbc/microsoft/sqlconnect-visual-foxpro-odbc-driver.md)<br /><br /> [SQLDriverConnect](../../odbc/microsoft/sqldriverconnect-visual-foxpro-odbc-driver.md)|  
|Su aplicación de Visual Basic|[Uso del controlador ODBC de Visual FoxPro con la aplicación de Visual Basic](../../odbc/microsoft/using-the-vfp-foxpro-odbc-driver-with-your-visual-basic-application.md)|
