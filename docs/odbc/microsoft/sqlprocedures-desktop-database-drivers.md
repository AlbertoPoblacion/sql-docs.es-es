# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures (controladores de escritorio de la base de datos)
**SQLProcedures** solo devolverá las filas para los procedimientos que tengan al menos un argumento. Se tratan los procedimientos que no tienen ningún argumento como vistas.  
  
|columna|Comentarios|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|Ruta de acceso al archivo de base de datos.|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|Nombre del procedimiento no delimitado|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
