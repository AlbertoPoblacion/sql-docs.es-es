# <a name="viewing-drivers"></a>Visualización de controladores
Para configurar orígenes de datos, debe haber instalado al menos un controlador en el sistema. La pestaña Controladores del cuadro de diálogo Administrador de **orígenes** de **datos ODBC** muestra todos los controladores instalados en el equipo, incluidos el nombre, la versión, la empresa, el nombre de archivo y la fecha de creación del archivo de cada controlador. Debe utilizar el programa de configuración del controlador para agregar o eliminar un controlador del sistema. Para obtener más información sobre cómo modificar controladores, consulte Administración de orígenes de [datos](../../odbc/admin/managing-data-sources.md).
