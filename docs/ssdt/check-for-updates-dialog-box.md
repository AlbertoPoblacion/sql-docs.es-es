# <a name="check-for-updates-dialog-box"></a>Cuadro de diálogo Buscar actualizaciones
A partir de la versión de noviembre de 2012 de SQL Server Data Tools, verá una opción de menú **Buscar actualizaciones** en el menú **SQL**.  
  
El cuadro de diálogo **Buscar actualizaciones** muestra la versión de SQL Server Data Tools instalada en el equipo y la versión publicada. Habrá disponible un botón **Actualizar** si la versión publicada es superior a la versión instalada. Haga clic en el botón **Actualizar** para ver una descripción de las actualizaciones disponibles y, si así lo decide, instalar la versión más reciente de SQL Server Data Tools en el equipo.  
  
El cuadro de diálogo **Buscar actualizaciones** también tiene una casilla **Buscar automáticamente actualizaciones de SQL Server Data Tools**. Cuando está activada, SQL Server Data Tools muestra una descripción de las actualizaciones disponibles cuando se inicia Visual Studio. Puede instalar esas actualizaciones.
