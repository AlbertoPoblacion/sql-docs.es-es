# <a name="rename-solutions-and-project-items"></a>Cambiar el nombre de las soluciones y los elementos de un proyecto
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
Es posible cambiar el nombre de una solución y de los elementos de proyecto en el Explorador de soluciones.  
  
> [!NOTE]  
> No es posible cambiar el nombre de los proyectos y las conexiones.  
  
### <a name="to-rename-a-solution-or-item"></a>Para cambiar el nombre de una solución o de un elemento  
  
1.  En el Explorador de soluciones, seleccione la solución o el elemento de proyecto cuyo nombre desea cambiar.  
  
2.  En el menú **Ver** , haga clic en **Ventana de propiedades**.  
  
3.  En la ventana Propiedades, modifique la propiedad **(Nombre)** .
