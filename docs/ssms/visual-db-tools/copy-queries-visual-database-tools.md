# <a name="copy-queries-visual-database-tools"></a>Copiar consultas (Visual Database Tools)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
Puede copiar el cuerpo de una consulta en otra nueva para utilizarlo como un punto de partida.  
  
### <a name="to-copy-a-query"></a>Para copiar una consulta  
  
1.  En el menú **Ver** , haga clic en Explorador de soluciones.  
  
2.  En el Explorador de soluciones, haga clic con el botón derecho en la consulta que desea copiar y elija **Abrir**en el menú contextual.  
  
3.  En el panel SQL, copie la parte de la consulta que desee.  
  
4.  Cree una nueva consulta o abra la consulta en la que desea pegar el código SQL copiado.  
  
5.  Coloque el cursor en el lugar en el que desea agregar el código SQL.  
  
6.  Haga clic con el botón derecho y, en el menú contextual, haga clic en **Pegar**.
