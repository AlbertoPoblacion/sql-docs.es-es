# <a name="show-diagram-properties-visual-database-tools"></a>Mostrar propiedades del diagrama (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Puede mostrar las propiedades de un diagrama de base de datos en la ventana Propiedades.  
  
### <a name="to-show-database-diagram-properties-in-the-properties-window"></a>Para mostrar las propiedades de los diagramas de base de datos en la ventana Propiedades  
  
1.  Abra el Diseñador de diagramas de base de datos.  
  
2.  Asegúrese de que no se ha seleccionado ningún objeto en el Diseñador de diagramas de base de datos; para ello, haga clic en cualquier lugar del diseñador excepto sobre un objeto.  
  
3.  En el menú **Ver** , haga clic en **Ventana Propiedades**.  
  
Podrá ver las propiedades del diagrama en la ventana **Propiedades** .
