# <a name="modify-queries-visual-database-tools"></a>Modificar consultas (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Abra una consulta para modificarla en el Diseñador de consultas y vistas.  
  
### <a name="to-open-a-query"></a>Para abrir una consulta  
  
1.  En el menú **Ver** , haga clic en **Explorador de soluciones**.  
  
2.  En el Explorador de soluciones, haga clic con el botón derecho en la consulta que desea abrir y, a continuación, haga clic en **Abrir**.  
  
3.  Para modificar la consulta en el Diseñador de consultas, resalte las instrucciones SQL, haga clic con el botón derecho en el área resaltada y seleccione **Diseñar consulta en el editor**.  
  
El Diseñador de consultas se abrirá y podrá realizar las modificaciones.
