# <a name="the-sql-server-management-studio-environment"></a>Entorno de SQL Server Management Studio
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
[!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] proporciona numerosas características para la administración y el desarrollo de consultas de [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)].  

> [!NOTE]
> Se han detectado conflictos con controladores de gráficos que causan problemas en el proceso de representación de SQL Server Management Studio y, como tal, la aceleración de gráficos de hardware está deshabilitada de forma predeterminada.  Si experimenta problemas de representación u otros problemas visuales en la interfaz de SSMS, compruebe que la opción "Usar aceleración de gráficos mediante hardware si está disponible" está *desactivada* en Herramientas > Opciones.
  
## <a name="in-this-section"></a>En esta sección  
[Elementos generales de la interfaz de usuario](../ssms/general-user-interface-elements.md)  
Esboza las características generales de la interfaz de usuario, como el Explorador de soluciones, el Explorador de objetos, la ventana Propiedades y el explorador web integrado.  
  
[Explorador de soluciones](../ssms/solution/solution-explorer.md)  
Presenta el Explorador de soluciones, que proporciona una vista organizada de los proyectos y sus archivos, así como un fácil acceso a los comandos necesarios para administrarlos.  
  
[Control de código fuente del Explorador de soluciones](https://docs.microsoft.com/sql/ssms/solution/solution-explorer)  
Describe cómo se integra [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] con los productos de control de código fuente.
