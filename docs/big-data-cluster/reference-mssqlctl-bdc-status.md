# <a name="mssqlctl-bdc-status"></a>estado de bdc mssqlctl

[!INCLUDE[tsql-appliesto-ssver15-xxxx-xxxx-xxx](../includes/tsql-appliesto-ssver15-xxxx-xxxx-xxx.md)]

El siguiente artículo proporciona la referencia para la **estado bdc** comandos en el **mssqlctl** herramienta. Para obtener más información acerca de otros **mssqlctl** comandos, consulte [mssqlctl referencia](reference-mssqlctl.md).

## <a name="commands"></a>Comandos:
|     |     |
| --- | --- |
[Mostrar estado de mssqlctl bdc](#mssqlctl-bdc-status-show) | Muestra el estado del clúster de datos grande.
## <a name="mssqlctl-bdc-status-show"></a>Mostrar estado de mssqlctl bdc
Muestra el estado del clúster de datos grande.
```bash
mssqlctl bdc status show 
```
### <a name="examples"></a>Ejemplos
Estado BDC donde el usuario ha iniciado sesión.
```bash
mssqlctl bdc status show
```
### <a name="global-arguments"></a>Argumentos globales
#### `--debug`
Aumenta el nivel de detalle de registro para mostrar todos los registros de depuración.
#### `--help -h`
Muestra este mensaje de ayuda y sale.
#### `--output -o`
Formato de salida.  Los valores permitidos: json, jsonc, table y tsv.  Predeterminado: json.
#### `--query -q`
Cadena de consulta de JMESPath. Consulte [ http://jmespath.org/ ](http://jmespath.org/]) para obtener más información y ejemplos.
#### `--verbose`
Aumenta el nivel de detalle de registro. Use --debug para obtener registros de depuración completos.
