# <a name="data-types-dmx"></a>Tipos de datos (DMX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Al utilizar extensiones de minería de datos (DMX) para definir un nuevo modelo de [!INCLUDE[msCoName](../includes/msconame-md.md)] minería de datos en [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)], debe proporcionar un tipo de datos para cada columna del modelo. El tipo de datos describe los datos que usará el algoritmo de minería de datos al generar el modelo.  
  
 Los tipos de datos dependen del algoritmo. Cada algoritmo determina los tipos de datos admitidos y su uso. Los algoritmos de Microsoft admiten los siguientes tipos de datos:  
  
-   Text  
  
-   long  
  
-   Boolean  
  
-   Double  
  
-   Date  
  
 Para obtener más información sobre los tipos de [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] datos que admite, vea [Data &#40;Types Data Mining&#41;](https://docs.microsoft.com/analysis-services/data-mining/data-types-data-mining).
