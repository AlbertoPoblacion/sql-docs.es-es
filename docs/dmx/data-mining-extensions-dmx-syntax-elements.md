# <a name="data-mining-extensions-dmx-syntax-elements"></a>Elementos de sintaxis de Extensiones de minería de datos (DMX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] En [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)], puede utilizar varios elementos de sintaxis para definir instrucciones DMX (extensiones de minería de datos) que puede utilizar para compilar, administrar y trabajar con modelos de minería de datos. En las secciones siguientes se describen estos elementos de sintaxis.  
  
## <a name="in-this-section"></a>En esta sección  
 [Identificadores &#40;DMX&#41;](../dmx/identifiers-dmx.md)  
 Asigne nombres a objetos como modelos de minería de datos, estructuras de minería de datos y columnas.  
  
 [Tipos de datos &#40;DMX&#41;](../dmx/data-types-dmx.md)  
 Definen el tipo de datos que contiene una columna de modelo de minería de datos.  
  
 [Expresiones &#40;DMX&#41;](../dmx/expressions-dmx.md)  
 Unidades de sintaxis que [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] puede resolver como valores de tabla, objetos o valores únicos o escalares.  
  
 [Operadores &#40;DMX&#41;](../dmx/operators-dmx.md)  
 Se usan con una o varias expresiones DMX para formar expresiones DMX más complejas.  
  
 [Funciones &#40;DMX&#41;](../dmx/functions-dmx.md)  
 Expresión que acepta cero o uno o varios valores de entrada, y que devuelve un valor escalar o una tabla.  
  
 [Comentarios &#40;DMX&#41;](../dmx/comments-dmx.md)  
 Elementos de texto que puede insertar en instrucciones o scripts DMX para explicar la finalidad de una instrucción. [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] no ejecuta comentarios.  
  
 [Palabras clave reservadas &#40;DMX&#41;](../dmx/reserved-keywords-dmx.md)  
 Son palabras reservadas para uso de DMX y que no deben emplearse como nombres de objetos de una base de datos.  
  
 [Tipos de contenido &#40;DMX&#41;](../dmx/content-types-dmx.md)  
 Define el contenido de una columna de estructura de minería de datos.  
  
 [Distribuciones &#40;DMX&#41;](../dmx/distributions-dmx.md)  
 Define la distribución de datos dentro de una columna.  
  
 [Uso &#40;DMX&#41;](../dmx/usage-dmx.md)  
 Define cómo usa un modelo de minería de datos las columnas que contiene.  
  
 [Marcas de modelado &#40;DMX&#41;](../dmx/modeling-flags-dmx.md)  
 Define sugerencias adicionales que puede usar el algoritmo para procesar un modelo de minería de datos.
