# <a name="postgresql-extension-preview"></a>Extensión de PostgreSQL (versión preliminar)

La extensión de PostgreSQL (versión preliminar) le permite conectarse a, consultar y desarrollar para Postgres mediante las funcionalidades de Azure Data Studio. 

Incluye la funcionalidad de Azure Data Studio disponible para PostgreSQL:

- Administrador de conexiones & editor de consultas
- Los paneles personalizables y widgets de datos
- Fragmentos de código
- [Terminal integrado](integrated-terminal.md)
- [Métodos abreviados de teclado](keyboard-shortcuts.md)
- [Integración de control de código fuente](source-control.md)
- [Configuración de área de trabajo y usuario](settings.md)


## <a name="install-the-postgresql-extension-preview"></a>Instalar la extensión de PostgreSQL (versión preliminar)

Si ya no tiene instalado Azure Data Studio, consulte su [las instrucciones de instalación](download.md).

1. Seleccione el icono de extensiones de la barra lateral en Azure Data Studio.
   ![Icono de extensiones](media/extensions/postgresql-extension/extensions-icon.png)

2. En la barra de búsqueda, escriba 'postgresql'. Seleccione la extensión de PostgreSQL.

3. Seleccione **Instalar**. Una vez instalado, seleccione **recarga** para activar la extensión en Azure Data Studio.
