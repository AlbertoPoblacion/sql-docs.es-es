# <a name="schema-compare-extension-preview"></a>Extensión Comparación de esquemas (versión preliminar)
La extensión de comparación de esquemas proporciona una experiencia fácil de usar para comparar las bases de datos y archivos .dacpac y aplicar los cambios del origen al destino.

Esta experiencia está actualmente en su versión preliminar inicial. Notificar problemas y solicitudes de características [aquí.](https://github.com/microsoft/azuredatastudio/issues)

## <a name="install-the-schema-compare-extension"></a>Instalar la extensión de comparación de esquemas

1. Para abrir el Administrador de extensiones y tener acceso a las extensiones disponibles, seleccione el icono de extensiones o **extensiones** en el **vista** menú.
2. Seleccione una extensión disponible para ver sus detalles.
1. Seleccione la extensión que desee (comparación de esquemas) y **instalar** lo.

## <a name="how-do-i-start-a-schema-comparison"></a>¿Cómo se puede iniciar una comparación de esquemas?
* Es el punto de entrada principal para la comparación de esquemas (ratón) en una base de datos en el Explorador de objetos y haga clic en **comparación de esquemas**.
* El usuario también puede iniciar el cuadro de diálogo de comparación de esquema de la paleta de comandos (Ctrl + Mayús + P) mediante la búsqueda de **comparación de esquemas**

## <a name="why-would-i-use-the-schema-compare"></a>¿Por qué se puede usar la comparación de esquemas?
Comparación de esquemas se creó para agregar la capacidad de comparar los esquemas de bases de datos y archivos .dacpac y aplicar los cambios.
