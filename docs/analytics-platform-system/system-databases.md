# <a name="system-databases-in-parallel-data-warehouse-pdw"></a>Bases de datos del sistema en almacenamiento de datos paralelos (PDW)
Las bases de datos del sistema son bases de datos que PDW de SQL Server utiliza para administrar usuarios, bases de datos y datos temporales. PDW de SQL Server incluye las siguientes bases de datos del sistema: Master y tempdb.  
  
## <a name="system-database-topics"></a>Temas de base de datos del sistema  
  
-   [Base de datos maestra](master-database.md)  
  
-   [Base de datos tempdb](tempdb-database.md)
