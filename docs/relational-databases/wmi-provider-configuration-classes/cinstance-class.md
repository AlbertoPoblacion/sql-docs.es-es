# <a name="cinstance-class"></a>Clase CInstance
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  La [clase CInstance](../../relational-databases/wmi-provider-configuration-classes/cinstance-class.md) representa una instancia de un cliente de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] de [!INCLUDE[msCoName](../../includes/msconame-md.md)].  
  
 Esta clase puede utilizarse para administrar los protocolos y las bibliotecas de red asociados a la instancia de un cliente de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].
