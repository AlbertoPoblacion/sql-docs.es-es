# <a name="cinstance-class"></a>Clase CInstance
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  La [clase CInstance](../../relational-databases/wmi-provider-configuration-classes/cinstance-class.md) representa una instancia de un [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] cliente.  
  
 Esta clase puede utilizarse para administrar los protocolos y las bibliotecas de red asociados a la instancia de un cliente de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].
