# <a name="serversettingsgeneralflag-class"></a>Clase ServerSettingsGeneralFlag
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  La [clase ServerSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) representa los valores de marca generales de una instancia de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Esta clase se puede utilizar para lo siguiente:  
  
-   Ver el nombre de la marca.  
  
-   Ver y modificar el valor de la marca.
