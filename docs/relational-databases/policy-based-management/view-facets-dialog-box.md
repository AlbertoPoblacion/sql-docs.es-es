# <a name="view-facets-dialog-box"></a>Ver facetas (cuadro de diálogo)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  Utilice este cuadro de diálogo para ver las propiedades de una faceta de administración basada en directivas. Las propiedades de una faceta se definen cuando se crea y no se pueden cambiar utilizando este cuadro de diálogo.  
  
## <a name="options"></a>Opciones  
 **Faceta**  
 Nombre de la faceta seleccionada actualmente.  
  
 **Descripción**  
 Proporciona una descripción general de la faceta.  
  
 **Propiedades de faceta**  
 Muestra los nombres y valores de las propiedades de faceta.  
  
 **Exportar estado actual como directiva**  
 Haga clic para exportar el estado descrito de las propiedades de faceta como una nueva directiva en un archivo XML.
