# <a name="smo-programming---getting-started-in-visual-c-net"></a>Programación de SMO: introducción a Visual C# .NET
[!INCLUDE [SQL Server ASDB, ASDBMI, ASDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asdw.md)]

  [!INCLUDE[csprcs](../../includes/csprcs-md.md)].NET es uno de los lenguajes de programación administrados que son compatibles con el Common Language Runtime (CLR) en [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] .net. Para obtener más información acerca del lenguaje C#, vea la documentación de [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] .NET. Puede utilizar [!INCLUDE[vsprvslong](../../includes/vsprvslong-md.md)] y [!INCLUDE[vsOrcas](../../includes/vsorcas-md.md)] para desarrollar aplicaciones SMO.  
  
## <a name="references-and-imports"></a>Referencias e importaciones  
 Una vez haya iniciado el proyecto de [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] , debe agregar las referencias a los ensamblados SMO. A continuación, importe los espacios de nombres necesarios para que su programa pueda reconocer los tipos SMO.  
  
 En [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] , los ensamblados SMO se encuentran en la carpeta c:\Archivos de programa\Microsoft SQL Server\130\SDK\Assemblies\
