# <a name="localdb_error_instance_stop_failed"></a>LOCALDB_ERROR_INSTANCE_STOP_FAILED
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
    
## <a name="details"></a>Detalles  
  
| Atributo | Value |
| --------- | ----- |
|Nombre de producto|SQL Server|  
|Id. de evento|271|  
|Origen de eventos|SQL Server Local Database Runtime 12.0|  
|Componente|API de Local Database Runtime|  
|Texto del mensaje|La operación de detención de la instancia de Local Database no se pudo completar en el tiempo especificado.|  
  
## <a name="explanation"></a>Explicación  
 La operación de detención no pudo completarse dentro del tiempo especificado.  
  
## <a name="user-action"></a>Acción del usuario  
 Intente de nuevo la operación solicitada.
