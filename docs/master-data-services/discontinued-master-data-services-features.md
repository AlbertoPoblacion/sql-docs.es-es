# <a name="discontinued-master-data-services-features"></a>Características descontinuadas de Master Data Services

[!INCLUDE [SQL Server - Windows only ASDBMI  ](../includes/applies-to-version/sql-windows-only-asdbmi.md)]

  Este tema describe las características de [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)] que ya no están disponibles en [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)].  
  
## <a name="sscurrent-discontinued-features"></a>[!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] Características no incluidas  
 El proceso de almacenamiento provisional de [!INCLUDE[ssKilimanjaro](../includes/sskilimanjaro-md.md)] se elimina en [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)]. Debe utilizar el proceso de almacenamiento provisional basado en entidades para importar y modificar datos. Para obtener más información, vea [información general: importar datos de tablas &#40;Master Data Services&#41;](../master-data-services/overview-importing-data-from-tables-master-data-services.md).  
  
 El modelo de metadatos ya no está disponible. Los usuarios tampoco pueden ver ya metadatos en el área funcional del Explorador.
