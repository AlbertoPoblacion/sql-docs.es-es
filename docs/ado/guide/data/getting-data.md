# <a name="getting-data"></a>Obtener datos
En concreto, los [conceptos básicos de ADO](../../../ado/guide/data/ado-fundamentals.md)y el ejemplo de [HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md) incorporaron las cuatro operaciones principales que intervienen en la creación de una aplicación ADO: obtención de datos, examen de datos, edición de datos y actualización de datos. En esta sección se explica cómo obtener datos con más detalle.  
  
 En un nivel básico, varios objetos ADO contribuyen a las operaciones de obtención de datos. En primer lugar, debe conectarse a un origen de datos mediante un objeto de **conexión** ADO. A continuación, se pasan instrucciones al origen de datos mediante un objeto **Command** de ADO. Por último, la mayoría de las veces recibe datos en un objeto de **conjunto de registros** ADO.
