# <a name="recordtype-property-ado"></a>Propiedad RecordType (ADO)
Indica el tipo de [registro](../../../ado/reference/ado-api/record-object-ado.md) objeto.  
  
## <a name="return-value"></a>Valor devuelto  
 Devuelve un [RecordTypeEnum](../../../ado/reference/ado-api/recordtypeenum.md) valor.  
  
## <a name="remarks"></a>Comentarios  
 El **RecordType** propiedad es de solo lectura.
