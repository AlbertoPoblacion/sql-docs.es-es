# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
Especifica el número de registros para recuperar desde un [Recordset](../../../ado/reference/ado-api/recordset-object-ado.md).  
  
|Constante|Valor|Descripción|  
|--------------|-----------|-----------------|  
|**adGetRowsRest**|-1|Recupera el resto de los registros de la **Recordset**, desde la posición actual o un marcador especificado por el *iniciar* parámetro de la [GetRows](../../../ado/reference/ado-api/getrows-method-ado.md) método.|

|--------------|
