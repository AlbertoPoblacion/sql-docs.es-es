# <a name="ado-code-examples-in-visual-basic"></a>Ejemplos de código ADO en Visual Basic

> [!NOTE]
>  Pegue el ejemplo de código completo, de sub a End Sub, en el editor de código. Es posible que el ejemplo no se ejecute correctamente si se usan ejemplos parciales o si se pierde el formato del párrafo.
