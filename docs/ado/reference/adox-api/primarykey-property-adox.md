# <a name="primarykey-property-adox"></a>PrimaryKey (propiedad, ADOX)
Indica si el índice representa la clave principal de la tabla.  
  
## <a name="settings-and-return-values"></a>Configuración y valores devueltos  
 Establece y devuelve un valor **booleano** .  
  
## <a name="remarks"></a>Observaciones  
 El valor predeterminado es **False**.  
  
 Esta propiedad es de solo lectura en los objetos de [Índice](../../../ado/reference/adox-api/index-object-adox.md) ya anexados a una colección.
