# <a name="append-method-adox-tables"></a>Append (método) (tablas ADOX)
Agrega un nuevo [tabla](../../../ado/reference/adox-api/table-object-adox.md) de objeto para el [tablas](../../../ado/reference/adox-api/tables-collection-adox.md) colección.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Tables.Append Table  
```  
  
#### <a name="parameters"></a>Parámetros  
 *Table*  
 Un **Variant** valor que contiene una referencia a la **tabla** para anexar o el nombre de la tabla para crear y anexar.  
  
## <a name="remarks"></a>Comentarios  
 Si el proveedor no admite la creación de tablas, se producirá un error.
