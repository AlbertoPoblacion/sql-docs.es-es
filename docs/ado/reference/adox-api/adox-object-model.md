# <a name="adox-object-model"></a>Modelo de objetos ADOX
En el diagrama siguiente se muestra cómo se representan los objetos y cómo se relacionan en ADOX. Para obtener más información acerca de un objeto o una colección concretos, vea el tema de referencia específico, o los [objetos ADOX](../../../ado/reference/adox-api/adox-objects.md) y [colecciones ADOX](../../../ado/reference/adox-api/adox-collections.md).  
  
 ![Modelo de objetos ADOX](../../../ado/reference/adox-api/media/adox_object_model.gif "ADOX_object_model")  
  
 Cada uno de los objetos [TABLE](../../../ado/reference/adox-api/table-object-adox.md), [index](../../../ado/reference/adox-api/index-object-adox.md)y [Column](../../../ado/reference/adox-api/column-object-adox.md) tiene también una colección de [propiedades](../../../ado/reference/ado-api/properties-collection-ado.md) de ADO estándar.
