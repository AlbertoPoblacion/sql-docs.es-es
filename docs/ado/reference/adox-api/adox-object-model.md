# <a name="adox-object-model"></a>Modelo de objetos ADOX
El siguiente diagrama ilustra cómo se representan y relacionados de ADOX objetos. Para obtener más información acerca de un objeto específico o una colección, vea el tema de referencia específica, o [objetos ADOX](../../../ado/reference/adox-api/adox-objects.md) y [colecciones ADOX](../../../ado/reference/adox-api/adox-collections.md).  
  
 ![Modelo de objetos ADOX](../../../ado/reference/adox-api/media/adox_object_model.gif "ADOX_object_model")  
  
 Cada uno de los [tabla](../../../ado/reference/adox-api/table-object-adox.md), [índice](../../../ado/reference/adox-api/index-object-adox.md), y [columna](../../../ado/reference/adox-api/column-object-adox.md) objetos también tiene un estándar ADO [propiedades](../../../ado/reference/ado-api/properties-collection-ado.md) colección.
