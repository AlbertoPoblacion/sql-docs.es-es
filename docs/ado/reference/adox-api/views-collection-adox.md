# <a name="views-collection-adox"></a>Colección de vistas (ADOX)
Todos los contiene [vista](../../../ado/reference/adox-api/view-object-adox.md) objetos de un catálogo.  
  
## <a name="remarks"></a>Comentarios  
 El [Append](../../../ado/reference/adox-api/append-method-adox-views.md) método para un **vistas** colección es único para ADOX. Puede:  
  
-   Agregar una nueva vista a la colección con el **Append** método.  
  
 Las propiedades y métodos restantes son estándar para colecciones de ADO. Puede:  
  
-   Obtener acceso a una vista de la colección con el [elemento](../../../ado/reference/ado-api/item-property-ado.md) propiedad.  
  
-   Devuelve el número de vistas incluido en la colección con el [recuento](../../../ado/reference/ado-api/count-property-ado.md) propiedad.  
  
-   Quitar una vista de la colección con el [eliminar](../../../ado/reference/adox-api/delete-method-adox-collections.md) método.  
  
-   Actualizar los objetos de la colección para reflejar el esquema de base de datos actual con el [actualizar](../../../ado/reference/ado-api/refresh-method-ado.md) método.
