# <a name="relatedtable-property-adox"></a>RelatedTable (propiedad, ADOX)
Indica el nombre de la tabla relacionada.  
  
## <a name="settings-and-return-values"></a>Configuración y valores devueltos  
 Establece y devuelve un valor de **cadena** .  
  
## <a name="remarks"></a>Observaciones  
 El valor predeterminado es una cadena vacía ("").  
  
 Si la clave es una clave externa, **RelatedTable** es el nombre de la tabla que contiene la clave.
