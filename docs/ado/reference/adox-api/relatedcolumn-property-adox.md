# <a name="relatedcolumn-property-adox"></a>RelatedColumn (propiedad, ADOX)
Indica el nombre del que se relaciona [objeto de columna (ADOX)](../../../ado/reference/adox-api/column-object-adox.md) en la tabla relacionada (solo columnas de clave).  
  
## <a name="settings-and-return-values"></a>Configuración y valores devueltos  
 Establece y devuelve un **cadena** valor que es el nombre de la columna relacionada de la tabla relacionada.  
  
## <a name="remarks"></a>Comentarios  
 El valor predeterminado es una cadena vacía ("").  
  
 Esta propiedad es de solo lectura para [columna](../../../ado/reference/adox-api/column-object-adox.md) ya anexados a una colección de objetos.
