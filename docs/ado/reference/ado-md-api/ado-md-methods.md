# <a name="ado-md-methods"></a>Métodos de ADO MD

|||  
|-|-|  
|[Cerrar](../../../ado/reference/ado-md-api/close-method-ado-md.md)|Cierra un conjunto de celdas abierto.|  
|[GetSchemaObject](../../../ado/reference/ado-md-api/getschemaobject-method-ado-md.md)|Recupera un objeto de esquema de ADO MD (dimensión, jerarquía, nivel o miembro) por un nombre único.|  
|[Abrir](../../../ado/reference/ado-md-api/open-method-ado-md.md)|Recupera los resultados de una consulta multidimensional y devuelve los resultados a un conjunto de celdas.|
