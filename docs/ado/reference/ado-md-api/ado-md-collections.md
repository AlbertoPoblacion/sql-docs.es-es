# <a name="ado-md-collections"></a>Colecciones de ADO MD

|||  
|-|-|  
|[Ejes](../../../ado/reference/ado-md-api/axes-collection-ado-md.md)|Contiene los objetos de **eje** que definen un objeto Cellset.|  
|[CubeDefs](../../../ado/reference/ado-md-api/cubedef-object-ado-md.md)|Contiene los objetos **CubeDef** que representan un cubo de un catálogo multidimensional.|  
|[Dimensiones](../../../ado/reference/ado-md-api/dimension-object-ado-md.md)|Contiene los objetos de **dimensión** que componen un cubo.|  
|[Jerarquías](../../../ado/reference/ado-md-api/hierarchy-object-ado-md.md)|Contiene los objetos de la **jerarquía** de conjunto de una dimensión.|  
|[Niveles](../../../ado/reference/ado-md-api/level-object-ado-md.md)|Contiene los objetos de **nivel** que componen una jerarquía.|  
|[Miembros](../../../ado/reference/ado-md-api/members-collection-ado-md.md)|Contiene los objetos **miembro** de un nivel o una posición a lo largo de un eje.|  
|[Positions](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)|Contiene los objetos de **posición** que definen un punto en un eje.|
