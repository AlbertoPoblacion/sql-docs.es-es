# <a name="ado-md-code-examples"></a>Ejemplos de código ADO MD
Utilice los siguientes ejemplos de código para obtener información sobre cómo usar los objetos, métodos y propiedades de ADO MD. Estos ejemplos son un subconjunto de las aplicaciones de ejemplo que se instala con servicios de OLAP de Microsoft SQL Server para SQL Server 7.0.  
  
 Estos ejemplos usa el proveedor OLE DB MSOLAP y ejecutarán en un host de servicios OLAP de Microsoft SQL Server local. Sin embargo, estos ejemplos están diseñados para mostrar técnicas fundamentales de programación de ADO MD y deben adaptarse con facilidad a otros orígenes de datos o los proveedores.  
  
-   [Ejemplos de código ADO MD en Visual Basic](../../../ado/reference/ado-md-api/ado-md-code-examples-in-visual-basic.md)  
  
-   [Ejemplos de código ADO MD en Visual Basic Scripting Edition](../../../ado/reference/ado-md-api/ado-md-code-examples-in-visual-basic-scripting-edition.md)
