# <a name="ado-md-enumerated-constants"></a>Constantes enumeradas de ADO MD
Para ayudar a la depuración, las constantes enumeradas de ADO MD muestran un valor para cada una de ellas. Sin embargo, este valor es puramente informativo y puede cambiar de una versión de ADO MD a otra. El código sólo debería depender del nombre, no el valor real, de constantes enumeradas.  
  
 Las siguientes enumeraciones se definen.  
  
-   [SchemaObjectTypeEnum](../../../ado/reference/ado-md-api/schemaobjecttypeenum.md)  
  
-   [MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md)
