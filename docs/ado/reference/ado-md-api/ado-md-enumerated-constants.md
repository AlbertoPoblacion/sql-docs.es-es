# <a name="ado-md-enumerated-constants"></a>Constantes enumeradas de ADO MD
Para ayudar a la depuración, los ADO MD constantes enumeradas muestran un valor para cada constante. Sin embargo, este valor es meramente consultivo y puede cambiar de una versión de ADO MD a otra. El código solo debe depender del nombre, no del valor real, de las constantes enumeradas.  
  
 Las enumeraciones siguientes están definidas.  
  
-   [SchemaObjectTypeEnum](./schemaobjecttypeenum.md)  
  
-   [MemberTypeEnum](./membertypeenum.md)
