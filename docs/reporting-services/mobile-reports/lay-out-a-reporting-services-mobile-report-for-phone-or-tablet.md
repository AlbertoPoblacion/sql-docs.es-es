# <a name="lay-out-a-reporting-services-mobile-report-for-phone-or-tablet"></a>Diseñar un informe para dispositivos móviles de Reporting Services para una tableta o teléfono
Ahora que ya ha [creado un informe para dispositivos móviles de Reporting Services](../../reporting-services/mobile-reports/create-a-reporting-services-mobile-report.md) en el [!INCLUDE[SS_MobileReptPub_Short](../../includes/ss-mobilereptpub-long.md)], puede diseñar una versión del mismo para teléfono o tableta.  
  
![SSMRP_SalesTrendRptLayout](../../reporting-services/mobile-reports/media/ssmrp-salestrendrptlayout.png)   
*Diseño principal del informe para dispositivos móviles.*  
  
1. En la esquina superior derecha del [!INCLUDE[SS_MobileReptPub_Short](../../includes/ss-mobilereptpub-short.md)], seleccione el icono de diseño >, ya sea **Teléfono** o **Tableta**.  
  
   ![SSMRP_LayoutMenu](../../reporting-services/mobile-reports/media/ssmrp-layoutmenu.png)  
     
   Se abre la cuadrícula de diseño de teléfono. Todas las visualizaciones se encuentran en la pestaña **Diseño** .  
     
   ![SSMRP_LayoutGrid](../../reporting-services/mobile-reports/media/ssmrp-layoutgrid.png)  
     
2. Arrastre las visualizaciones a la cuadrícula y cámbielas de tamaño.  
  
   ![SSMRP_PhoneLayout](../../reporting-services/mobile-reports/media/ssmrp-phonelayout.png)  
     
3. Seleccione **Vista previa** para ver cómo queda el informe en un teléfono.  
  
   ![SSMRP_PhonePreview](../../reporting-services/mobile-reports/media/ssmrp-phonepreview.png)
