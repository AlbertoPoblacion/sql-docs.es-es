# <a name="master-data-services-server-configuration-page-configuration-manager"></a>Página Configuración del servidor de Master Data Services (Administrador de configuración)
  Utilice la página **Configuración del servidor** para ver información de configuración del equipo del servidor local.  
  
## <a name="server-configuration"></a>Configuración del servidor  
  
|Nombre del control|Descripción|  
|------------------|-----------------|  
|**Windows PowerShell**|Muestra la versión de Windows PowerShell que se instala en el equipo local. Se requiere la versión 1.0 o posterior de Windows PowerShell para utilizar el Administrador de configuración de Master Data Services.|  
|**Internet Information Services**|Muestra la versión de Internet Information Services (IIS) que se instala en el equipo local. Se requiere la versión 7.0 o posterior de IIS para utilizar el Administrador de configuración de Master Data Services.|
