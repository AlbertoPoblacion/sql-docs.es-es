# <a name="sort-multiple-columns-in-queries-visual-database-tools"></a>Ordenar varias columnas en las consultas (Visual Database Tools)
  Puede especificar el orden en el que desea usar las columnas para ordenar los datos utilizando la columna **Criterio de ordenación** en el [panel Criterios](visual-database-tools.md).  
  
### <a name="to-sort-query-results-by-more-than-one-column"></a>Para ordenar los resultados de la consulta por más de una columna  
  
1.  En el panel **Criterios** , haga clic en el campo **Criterio de ordenación** de la columna por la que desea realizar la ordenación.  
  
2.  Escriba un valor entero o selecciónelo en la lista desplegable para indicar el lugar que ocupa la columna en el orden de columnas ordenadas.  
  
     Repita los pasos para cada columna por la que desee ordenar los resultados.
