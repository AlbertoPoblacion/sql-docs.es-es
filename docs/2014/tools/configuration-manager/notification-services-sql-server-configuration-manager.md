# <a name="notification-services-sql-server-configuration-manager"></a>Notification Services (Administrador de configuración de SQL Server)
  Este es el servicio que admite [!INCLUDE[ssNS](../../includes/ssns-md.md)]. Para ver las propiedades del servicio, en el panel de detalles, haga clic con el botón derecho en el servicio y, después, haga clic en **Propiedades**.  
  
 Para obtener información acerca de [!INCLUDE[ssNS](../../includes/ssns-md.md)], vea los Libros en pantalla.
