# <a name="summary-writing-transact-sql"></a>Resumen: Escribir Transact-SQL
  Editor de consultas es un paso importante hacia un entorno de creación más sofisticado en [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] desarrollo. Puede cambiar con facilidad el Editor de consultas al modo de pantalla completa para ver el código o cambiar rápidamente la conexión a otro servidor. Es un buen lugar para buscar características adicionales en el **opciones** cuadro de diálogo, disponible desde el **herramientas** menú.
