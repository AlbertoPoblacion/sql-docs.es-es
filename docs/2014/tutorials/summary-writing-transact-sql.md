# <a name="summary-writing-transact-sql"></a>Resumen: Escribir Transact-SQL
  El editor de consultas es un paso importante hacia un entorno de creación más [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] sofisticado en el desarrollo. Puede cambiar con facilidad el Editor de consultas al modo de pantalla completa para ver el código o cambiar rápidamente la conexión a otro servidor. Un buen lugar para buscar características adicionales es en el cuadro de diálogo **Opciones** , disponible en el menú **herramientas** .
