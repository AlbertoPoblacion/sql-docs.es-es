# <a name="updategram-sample-applications-sqlxml-40"></a>Aplicaciones de ejemplo de diagrama de actualización (SQLXML 4.0)
  En esta sección se proporcionan ejemplos del uso de diagramas de actualización.  
  
 Todos los ejemplos de esta sección usan la base de datos de [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]ejemplo AdventureWorks en. Todas las actualizaciones se aplican a las tablas de la base de datos AdventureWorks.

 [Ejecutar un diagrama mediante OLE DB &#40;SQLXML 4,0&#41;](../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/updategrams/executing-an-updategram-by-using-ole-db-sqlxml-4-0.md)  
 Proporciona un ejemplo funcional del uso de OLE DB para ejecutar un diagrama de actualización.  
  
 [Usar un diagrama en una aplicación ASP de ejemplo &#40;SQLXML 4,0&#41;](../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/updategrams/using-an-updategram-in-a-sample-asp-application-sqlxml-4-0.md)
