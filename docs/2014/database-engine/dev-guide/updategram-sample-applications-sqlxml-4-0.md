# <a name="updategram-sample-applications-sqlxml-40"></a>Aplicaciones de ejemplo de diagrama de actualización (SQLXML 4.0)
  En esta sección se proporcionan ejemplos del uso de diagramas de actualización.  
  
 Todos los ejemplos en esta sección usan la base de datos de ejemplo AdventureWorks en [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Todas las actualizaciones se aplican a las tablas de la base de datos AdventureWorks.  
  
## <a name="in-this-section"></a>En esta sección  
 [Ejecutar un diagrama de actualización mediante ADO &#40;SQLXML 4.0&#41;](../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/updategrams/executing-an-updategram-by-using-ado-sqlxml-4-0.md)  
 Proporciona una aplicación de [!INCLUDE[msCoName](../../includes/msconame-md.md)] Visual Basic que usa ADO para establecer una conexión a [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] y ejecutar un diagrama de actualización.  
  
 [Ejecutar un diagrama de actualización mediante OLE DB &#40;SQLXML 4.0&#41;](../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/updategrams/executing-an-updategram-by-using-ole-db-sqlxml-4-0.md)  
 Proporciona un ejemplo funcional del uso de OLE DB para ejecutar un diagrama de actualización.  
  
 [Uso de un diagrama de actualización en una aplicación de ASP de ejemplo &#40;SQLXML 4.0&#41;](../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/updategrams/using-an-updategram-in-a-sample-asp-application-sqlxml-4-0.md)  
 Proporciona una aplicación ASP (Páginas Active Server) para actualizar la información del cliente en la tabla Person.Contact de la base de datos AdventureWorks.
