# <a name="installationid-property-wmi-msreportserverconfigurationsetting"></a>Propiedad InstallationID (WMI MSReportServer_ConfigurationSetting)
  Esta propiedad devuelve un identificador único para una instancia del servidor de informes concreta.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
