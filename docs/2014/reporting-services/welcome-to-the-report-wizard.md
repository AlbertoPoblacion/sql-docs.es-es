# <a name="welcome-to-the-report-wizard"></a>Página de presentación del Asistente para informes
  Utilice esta página del Asistente para informes para ver un resumen de las tareas que se van a realizar.  
  
## <a name="options"></a>Opciones  
 **No volver a mostrar esta página**  
 Seleccione esta opción para omitir esta página la próxima vez que se inicie el asistente.
