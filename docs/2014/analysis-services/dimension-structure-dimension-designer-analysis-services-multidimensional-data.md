# <a name="dimension-structure-dimension-designer-analysis-services---multidimensional-data"></a>Estructura de dimensión (Diseñador de dimensiones) (Analysis Services - Datos multidimensionales)
  Use la página **Estructura de dimensión** del Diseñador de dimensiones para modificar los atributos, las jerarquías y los niveles de la dimensión seleccionada. La página **Estructura de dimensión** contiene una barra de herramientas y tres paneles.  
  
 **Para mostrar la página de la estructura de dimensión**  
  
-   En [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)], abra un proyecto de [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] y, a continuación, abra la dimensión que desea ver.  
  
     De forma predeterminada, la página **Estructura de dimensión** aparece al abrir primero una dimensión en el Diseñador de dimensiones.  
  
## <a name="toolbar"></a>Barra de herramientas  
 Use la barra de herramientas para realizar acciones comunes en la página **Estructura de dimensión** del Diseñador de dimensiones. Para más información sobre este panel, vea [Barra de herramientas &#40;pestaña Estructura de dimensiones, Diseñador de dimensiones&#41; &#40;Analysis Services - Datos multidimensionales&#41;](toolbar-dimension-structure-designer-analysis-services-multidimensional-data.md).  
  
## <a name="panes"></a>Paneles  
 La página **Estructura de dimensión** contiene los siguientes paneles:  
  
 [Vista del origen de datos](datasource-view-dimension-designer-analysis-services-multidimensional-data.md)  
 Use el panel **Vista del origen de datos** para explorar y administrar la vista del origen de datos asociada a una dimensión.  
  
 [Jerarquías](hierarchies-dimension-designer-analysis-services-multidimensional-data.md)  
 Use el panel **Jerarquías** para administrar las jerarquías y los niveles de una dimensión.  
  
 [Atributos](attributes-dimension-designer-analysis-services-multidimensional-data.md)  
 Use el panel **Atributos** para explorar y administrar los atributos de una dimensión.
