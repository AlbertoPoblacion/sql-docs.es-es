# <a name="mining-models-view-data-mining-model-designer"></a>Modelos de minería de datos (vista del Diseñador de modelos de minería de datos)

 La vista **Modelos de minería de datos** tiene las siguientes herramientas y cuadros de diálogo:  
  
-   [Cuadro de diálogo parámetros de algoritmo &#40;vista los modelos de minería de datos&#41;](algorithm-parameters-dialog-box-mining-models-view.md)
