# <a name="complete---complete-image"></a>Completar - Completar imagen
  Use la página **Completar** para comprobar que la configuración de la instancia preparada se ha completado.  
  
## <a name="options"></a>Opciones  
 Este cuadro de diálogo incluye un vínculo al archivo de registro del programa de instalación para esta operación. Para completar la instalación, haga clic en **Cerrar**.
