# <a name="complete---upgrade"></a>Completar - Actualizar
  Use la página Completada (Actualizar) para comprobar que la actualización se ha completado.  
  
## <a name="options"></a>Opciones  
 Este cuadro de diálogo incluye un vínculo al archivo de registro del programa de instalación para esta operación. Para continuar, haga clic en **Cerrar**.
