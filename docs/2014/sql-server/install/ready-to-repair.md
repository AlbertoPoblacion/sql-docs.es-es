# <a name="ready-to-repair"></a>Listo para reparar
  La página Listo para reparar proporciona un resumen del proceso de reparación por parte del programa de instalación de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Opciones  
 En esta página dispone de las siguientes opciones:  
  
-   Para continuar con la instalación, haga clic en **Reparar**.  
  
-   Para cambiar la configuración de la instalación, haga clic en **Atrás**.  
  
-   Para salir de la instalación, haga clic en **Cancelar**.
