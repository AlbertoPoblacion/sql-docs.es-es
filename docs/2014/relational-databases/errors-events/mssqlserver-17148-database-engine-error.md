# <a name="mssqlserver_17148"></a>MSSQLSERVER_17148
    
## <a name="details"></a>Detalles  
  
|Atributo|Value|  
|-|-|  
|Nombre de producto|SQL Server|  
|Id. de evento|17148|  
|Origen de eventos|MSSQLSERVER|  
|Componente|SQLEngine|  
|Nombre simbólico|STARTUP_SERVERUNINSTALL|  
|Texto del mensaje|Se está cerrando SQL Server en respuesta a una solicitud 'stop' del Administrador de control de servicios. Esto es solo un mensaje informativo. No se requiere ninguna acción del usuario.|  
  
## <a name="explanation"></a>Explicación  
 El Administrador de control de servicios ha cerrado SQL Server  
  
## <a name="user-action"></a>Acción del usuario  
 None
