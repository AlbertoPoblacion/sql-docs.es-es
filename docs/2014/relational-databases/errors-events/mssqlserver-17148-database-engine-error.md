# <a name="mssqlserver17148"></a>MSSQLSERVER_17148
    
## <a name="details"></a>Detalles  
  
|||  
|-|-|  
|Nombre del producto|SQL Server|  
|Identificador del evento|17148|  
|Origen del evento|MSSQLSERVER|  
|Componente|SQLEngine|

|Texto del mensaje|Se está cerrando SQL Server en respuesta a una solicitud 'stop' del Administrador de control de servicios. Esto es solo un mensaje informativo. No se requiere ninguna acción del usuario.|
