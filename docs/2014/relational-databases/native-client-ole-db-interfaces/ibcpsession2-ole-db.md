# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  La interfaz de IBCPSession2 es una extensión IBCPSession que proporciona una función miembro que es una alternativa a llamar a ibcpsession:: BCPColFmt para cada columna.  IBCPSession2 hereda IBCPSession y agrega un nuevo método: [IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md).
