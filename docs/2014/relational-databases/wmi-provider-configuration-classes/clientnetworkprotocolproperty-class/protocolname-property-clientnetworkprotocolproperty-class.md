# <a name="protocolname-property-clientnetworkprotocolproperty-class"></a>Propiedad ProtocolName (clase ClientNetworkProtocolProperty)
  Obtiene el nombre del protocolo que posee la propiedad actual al que hace referencia el [Propertyidx (clase ClientNetworkProtocolProperty)](clientnetworkprotocolproperty-class.md) valor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Un [clase ClientNetworkProtocolProperty](clientnetworkprotocolproperty-class.md) objeto que representa un atributo del protocolo de red utilizado por el [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] cliente.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el nombre del protocolo que posee la propiedad.
