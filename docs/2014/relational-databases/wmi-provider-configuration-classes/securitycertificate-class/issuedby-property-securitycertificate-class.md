# <a name="issuedby-property-securitycertificate-class"></a>Propiedad IssuedBy (clase SecurityCertificate)
  Obtiene el objeto que emitió el certificado de seguridad.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.IssuedBy [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase SecurityCertificate](securitycertificate-class.md) que representa un certificado de seguridad.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el objeto que emitió el certificado de seguridad.
