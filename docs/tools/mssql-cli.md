# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>herramienta de consulta de línea de comandos de MSSQL-CLI para SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

MSSQL-CLI es una herramienta de línea de comandos interactiva para consultar SQL Server, instalarla en Windows, macOS o Linux.

## <a name="install-mssql-cli"></a>Instalación de MSSQL-CLI

Para obtener instrucciones de instalación detalladas, consulte la [Guía de instalación](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md), o bien, si conoce PIP, ejecute el siguiente comando para instalar:

```$ pip install mssql-cli```
